# 邀请中心

邀请中心用于查看邀请进展、追踪被邀请人状态，以及管理账户的邀请关系绑定。

![](./images/I9NfbL1qroE2GlxfpCCjHvTUpEf.png)

邀请中心

## 活动状态

目前为加速实现“AI券商”的战略愿景，Longbridge 正对账户系统进行全面升级。此次升级不仅提升开户审核效率，更将为未来一系列智能化功能与账号关联，给用户带来更优的体验。为确保系统平稳切换，优化期间暂不受理新客户开户申请，现有客户账户与交易服务均不受影响。我们正积极推进升级工作，完成后将尽快恢复开户服务。感谢您的理解与支持。

## 页面入口

「我的」→「邀请中心」

## 我的邀请码

页面展示个人专属邀请码，点击可复制分享给好友。好友通过邀请码注册并完成相应操作后，双方可获得活动奖励。

## 被邀请人列表

展示通过本账户邀请注册的所有用户，每条记录包含：

| 字段 | 说明 |
| --- | --- |
| 用户信息 | 被邀请人昵称（脱敏显示） |
| 联系方式 | 注册手机号（部分隐藏） |
| 进度状态 | 注册 → 开户 → 资产达标，每步附完成日期 |

资产达标状态每日 20:00 更新。

## 累计奖励

页面顶部显示通过邀请活动累计获得的奖励金额，支持切换查看不同货币的奖励统计。点击「奖励明细」查看每笔奖励的详细记录。

## 搜索

支持按昵称搜索被邀请人，快速查找特定好友的邀请进度。

## 修改邀请关系

![](./images/VeWfbTXwdo8SqzxiVxqjTZyipK2.png)

修改邀请关系

进入「绑定或修改邀请关系」页面管理账户的邀请绑定状态。

邀请关系分为三个阶段：

| 邀请关系 | 说明 | 是否可修改 |
| --- | --- | --- |
| 注册邀请人 | 注册时绑定的邀请人 | 注册后不可修改 |
| 开户邀请人 | 开户时绑定的邀请人 | 开户完成后不可修改 |
| 入资邀请人 | 首次入资时绑定的邀请人 | 入资前可主动搜索绑定；入资后锁定 |

未绑定入资邀请人时，可在该页面主动搜索长桥用户并绑定。
