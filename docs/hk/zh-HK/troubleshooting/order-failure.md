# 订单提交失败怎么办

下单时提示错误或订单无法提交，根据错误信息或现象定位原因。

## 账户资金不足

提示「资金不足」「余额不足」「Insufficient funds」：

1. 进入「资产」页面确认可用余额，注意区分币种（港元 / 美元）
2. 如委托金额超出可用余额，降低数量或金额后重新下单
3. 如账户有余额但仍提示不足：检查是否有其他订单占用了可用资金（已挂出的未成交买单会占用资金）
4. 如需更多资金，通过入金或换汇补充

## 交易时间限制

提示「市场已关闭」「非交易时段」：

| 市场 | 正常交易时间 |
|-----|------------|
| 港股（持续交易） | 交易日 09:30–12:00，13:00–16:00（HKT） |
| 港股竞价 | 开市竞价 09:00–09:20，收市竞价 16:00–16:10 |
| 美股（正式时段） | 美东 09:30–16:00 |
| 美股盘前 | 美东 04:00–09:30 |
| 美股盘后 | 美东 16:00–20:00 |
| 美股期权 | 美东 09:30–16:00（部分 ETF 期权至 16:15） |

港股盘前盘后暂不支持交易。如在交易时间内仍提示市场关闭，检查标的是否正在停牌或暂停交易。

## 标的不支持或权限不足

提示「该标的不支持交易」「权限不足」：

- **港股碎股**：碎股只能使用「碎股单」类型，不支持市价单
- **美股期权**：确认账户已开通期权交易权限；部分流通性较低的期权合约暂不支持
- **卖空**：如提示无法卖空，确认该标的支持融券且账户已开通融资账户
- **期权盘前**：仅部分热门标的支持盘前期权交易，在期权链页面勾选「支持盘前交易」筛选

## 保证金不足（融资 / 期权卖方）

提示「保证金不足」「Margin insufficient」：

1. 进入「资产」确认保证金使用率
2. 降低订单数量，使保证金要求降至可用范围内
3. 或先平仓部分持仓释放保证金后再下单

如账户触发 Margin Call，新开仓可能被限制，需先补充资金或平仓。

## 订单参数有误

提示「价格超出范围」「最小手数」「委托数量无效」：

- **限价单价格偏离**：港股限价单价格须在最新成交价的一定范围内（通常 ±24 个价格档位）；过高或过低的价格会被拒绝
- **港股手数限制**：整手订单数量须为一手的整数倍（如一手 100 股，则数量须为 100、200、300 等）
- **最小金额**：美股定投最低 10 美元；部分产品有最低委托金额要求

## 网络或系统问题

下单按钮无响应，或提示「提交失败」「网络错误」：

1. 检查网络连接，切换 Wi-Fi / 移动数据后重试
2. 关闭 VPN 或代理后重试
3. 强制重启 App 后重新下单
4. 如问题持续，参阅 [APP 网络连接排查](/troubleshooting/network-connection)

## 股票停牌或暂停交易

如标的正在停牌（港股 / 美股），系统会拒绝新订单直到复牌。可在标的详情页或公告中确认停牌状态。

## 仍无法解决

联系客服时提供：

- 错误提示截图
- 标的代码（如 00700、AAPL）
- 订单类型、数量和价格

**在线客服**：长桥 App → 我的 → 帮助与客服 → 在线客服
