# 交易收费

长桥香港账户的默认标准交易费率，涵盖港股、美股、期权及融资收费，并附佣金计算示例。

## 港股市场

交易佣金：0.03% × 交易金额，单笔最低 3 HKD（持有终身免佣权益时永久免佣金） 平台费：15 HKD / 笔

第三方代收费用（按已成交交易金额计算）：

| 费用项目 | 收费标准 |
| --- | --- |
| 交收费 | 0.0042% × 交易金额 |
| 印花税 | 0.1% × 交易金额（不足 1 港元按 1 港元计，涡轮牛熊和 ETF 不收取） |
| 交易费 | 0.00565% × 交易金额（最低 0.01 HKD） |
| 交易征费 | 0.0027% × 交易金额（最低 0.01 HKD） |
| 财汇局交易征费 | 0.00015% × 交易金额（最低 0.01 HKD） |

## 美股市场

交易佣金：0.0049 USD / 股，单笔最低 0.99 USD（持有终身免佣权益时永久免佣金） 平台费：0.005 USD / 股，单笔最低 1 USD

保护规则：佣金最多收取交易金额的 0.5%，平台使用费最多收取交易金额的 0.99%。若与每笔最低收费标准冲突，以最低收费标准为准。

第三方费用：

| 费用项目 | 收费标准 |
| --- | --- |
| 交收费 | 0.003 USD / 股，最高 7% * 交易金額 |
| SEC 费用 | 0.00206% * 交易金額，最低 0.01 USD |
| TAF 交易活动费 | 0.000195 USD x 賣出數量，最低 0.01 USD，最高 9.79 USD |
| CAT 综合审计追踪监管费 | 0.000003 USD / 股，最低 0.01 USD（自 2026 年 5 月 1 日起收取） |

### 美股碎股交易收费

| 单笔订单成交数量 | 佣金 | 平台使用费 |
| --- | --- | --- |
| < 1 股 | 免费 | 0.99% × 交易金额，每笔订单最高 0.99 USD |
| ≥ 1 股 | 按美股收费规则正常收取 | 按美股收费规则正常收取 |

## 美股期权

交易佣金：

| 期权权利金 | 佣金 |
| --- | --- |
| > 0.10 USD | 0.45 USD / 张，单笔最低 1.49 USD |
| ≤ 0.10 USD | 0.10 USD / 张，单笔最低 0.99 USD |

平台费：0.30 USD / 张

第三方费用：

| 费用项目 | 收费标准 |
| --- | --- |
| 交易活动费（TAF） | 0.00329 USD / 張，單筆訂單最低 0.01 USD（仅卖单收取） |
| 期权监管费 | 0.02295 USD / 張 |
| 期权清算费 | 0.025 USD / 张 |
| 期权交收费 | 0.18 USD / 张 |
| CAT 综合审计追踪监管费 | 0.0003 USD / 张，每次成交最低 0.01 USD（自 2026 年 5 月 1 日起收取） |

当期权被卖出行使时，会收取美股卖出股票的相关第三方费用。

## 融资收费

| 货币 | 年利率 |
| --- | --- |
| 港元 | 5.75% |
| 美元 | 5.75% |

利息公式：融资金额 × 利率 × 实际融资天数 ÷ 365，最低 0.01 港元/美元

融资天数按自然日计算。账户现金结余为负数时产生利息（持仓过夜收息，日内还款不收利息）。利息当月月终收取。

以上为默认标准融资利率，实际个人融资利率请在长桥 App - 我的 - 我的费率 查看。

## 打新 / 暗盘

港股打新手续费：

| 类型 | 费用 |
| --- | --- |
| 普通认购手续费 | 49 HKD |
| 孖展认购手续费 | 99 HKD |
| 中签收费 | 中签金额的 1.0085%（由香港证监会、交易所、经纪公司收取，仅在中签时收取） |

暗盘交易收费：

| 费用项 | 收费标准 |
| --- | --- |
| 佣金 | 0.03% × 交易金额，单笔最低 3 HKD |
| 平台使用费 | 15 HKD / 笔 |

暗盘代收第三方费用与港股市场一致（交收费、印花税、交易费、交易征费、财汇局征费）。

暗盘交易收费与普通港股交易收费套餐及佣金状态无关，按暗盘交易收费标准正常收取。

| 项目 | 费率 |

| 美股股息税 | 股息是否征收预扣税及其适用税率，主要取决于公司注册地的税法规定  |

## 个人费率查询

每位用户的实际佣金、平台费和融资利率可能因账户类型、活动优惠、免佣卡等因素不同。以上为默认标准费率，仅供参考。

查询用户的个人实际费率（佣金、平台费、融资利率），向用户说明其个人适用的费率。

查看个人费率：长桥 App - 我的 - 我的费率

## 通用规则

- 交易佣金和平台费按每笔订单单独计算，不合并

- 未交割交易免息，交割当天开始计息

- 终生免佣规则：卡券激活次日生效，免除对应市场非暗盘交易的佣金
