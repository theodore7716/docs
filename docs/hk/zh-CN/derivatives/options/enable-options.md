# 开通期权交易

长桥美股期权交易需要单独开通权限。已持有香港或新加坡综合账户的用户均可申请。

## 开通步骤

1. 打开长桥 App，进入「资产」页面
2. 点击右上角「全部功能」
3. 找到「开通功能」或「开户资料」入口，选择「美股期权」
4. 阅读并确认期权交易风险披露书
5. 完成适当性评估问卷（含投资经验、风险承受能力等题目）
6. 提交申请，等待审核

审核通常在 1 个工作日内完成，审核结果会通过 App 通知推送。

## 权限级别

长桥按风险程度提供分级期权权限：

| 级别 | 可操作策略 |
|------|----------|
| 一级 | 买入看涨（Long Call）、买入看跌（Long Put）—— 仅需支付权利金 |
| 二级 | 一级 + 有持仓担保的卖出期权（Covered Call、Cash-secured Put） |
| 三级 | 二级 + 裸卖出期权（Naked Call / Naked Put）—— 需要较高保证金 |

新用户通常先获得一级权限。如需升级，可在 App 内重新提交申请并更新适当性评估。

## 开通失败与审核答疑

**Q：提交申请后多久可以开始交易？**  
审核通过后即可开始交易。审核通常在提交当日或次日完成（工作日）。

**Q：申请被拒怎么办？**  
拒绝原因通常是适当性评估分数不足或账户条件未满足。可联系客服了解具体原因，部分情况下可补充说明后重新提交。

**Q：港股窝轮和牛熊证需要单独申请吗？**  
不需要，港股窝轮和牛熊证属于正股交易范畴，持有香港账户即可直接交易，无需额外开通权限。
