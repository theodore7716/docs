# 标准账户出金

标准账户出金支持的币种、方式、到账时间、各币种手续费及限制条件。

## 重要提示

长桥已于 2025 年 5 月 31 日停止长桥标准账户的服务。长桥标准账户目前仅支持卖出、出金或转出操作。

**长桥新西兰（LBNZ）标准账户停服说明（2025年5月31日起）：** 长桥新西兰标准账户已于2025年5月31日停止服务，新西兰账户不再支持入金。账户仅支持卖出持仓、出金或资产转出操作。出金支持多种币种（HKD/USD/AUD/CNY/MYR/SGD/GBP/EUR/JPY），各币种出金手续费由长桥方收取（如HKD 65/笔、USD 7.5/笔等），请以实际扣费为准。

## 支持币种

港元（HKD）、美元（USD）、澳元（AUD）、人民币（CNY）、马来西亚令吉（MYR）、新加坡元（SGD）、英镑（GBP）、欧元（EUR）、日元（JPY）

## 支持出金方式

电汇（WIRE）、网银转账、FPS 转数快

## 到账时间

预计 1-3 个交易日

## 手续费

**注意：出金手续费不适用退款。**

长桥按笔收取手续费，各币种标准如下：

| 币种 | 手续费 |
| --- | --- |
| HKD | 65 HKD / 笔 |
| USD | 7.5 USD / 笔 |
| AUD | 10 AUD / 笔 |
| CNH | 50 CNH / 笔 |
| MYR | 30 MYR / 笔 |
| SGD | 5 SGD / 笔 |
| GBP | 5.5 GBP / 笔 |
| EUR | 7 EUR / 笔 |
| JPY | 1,200 JPY |

 汇出行、中转行及收款行可能额外收取费用，最终到账金额以实际为准。

### 特殊收费说明

- 非新加坡本地银行卡及新加坡本地银行出金美元：4 USD / 笔（支票方式）

- SGD 出金到新加坡本地银行卡：提现金额不得低于 1 新币

- SGD 出金到香港本地银行：10 SGD / 笔

- 美元出金至香港本地银行：总手续费约 15 USD（上游券商 7.5 USD + 汇出行/中转行/收款行约 7-8 USD）

- 港币出金至不支持 FPS 的香港本地银行：65 HKD + 60-80 HKD（银行处理费用）

- 出金至华美银行：7.5 USD + 40 USD（银行处理费用）

## 操作流程

长桥 App - 资产 - 全部功能 - 提取资金，选择币种、收款银行、输入金额后提交。

## 支持账户类型与出金限制

- 仅支持本人同名银行卡出金

- 不支持 SIB、CIB、Discover Bank 出金

- 不支持信用卡出金

- 不支持 Wise 出入金

- 出金收费包含券商手续费及银行费用，最终到账金额与提现金额可能有差异
