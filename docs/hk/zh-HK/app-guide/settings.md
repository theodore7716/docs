# App 设置

设置是 App 的全局配置中心，提供账号安全、交易偏好、通知、隐私等各类个性化配置项。

## 页面入口

「我的」→「设置」

## 通用

| 设置项       | 说明                 |
|-----------|--------------------|
| 皮肤设置      | 选择 App 显示主题（如经典黑）  |
| 语言设置      | 切换 App 界面语言        |
| 屏幕常亮      | 开启后屏幕不自动熄灭         |
| 截屏快捷分享    | 截屏后快速分享            |
| 自动下载最新安装包 | 仅 Wi-Fi 网络下自动下载更新包 |
| 辅助功能      | 无障碍相关设置            |
| 切换接入点     | 切换服务器接入节点          |
| 网络诊断      | 检测当前网络连接状态         |
| 清理缓存      | 显示当前缓存大小，点击清理      |

## 账号与安全

**账号信息：**

- 登录 ID
- 手机号（可修改）
- 邮箱（可修改）

> 修改接收结单、交易通知的手机号/邮箱，需前往「账户设置 → 联系方式」。

**第三方账号绑定：**

- 微信
- 支付宝

**安全设置：**

| 设置项    | 说明                   |
|--------|----------------------|
| 设备管理   | 查看和管理已登录设备           |
| 修改登录密码 | 更改账号登录密码             |
| 生物识别登录 | 开启指纹或面容 ID 登录        |
| 登录有效时长 | 设置登录状态的保持时长（默认 30 天） |
| 企业员工认证 | 认证长桥企业员工身份           |
| 注销账号   | 永久注销账号               |

## 账户设置

账户设置针对具体证券账户，支持切换账户分别配置。

| 设置项      | 说明                                 |
|----------|------------------------------------|
| 折算货币     | 设置资产总览的折算显示货币（默认 USD）              |
| 修改交易密码   | 修改当前账户的交易密码                        |
| 交易密码有效时长 | 输入一次交易密码后的免输有效期（默认 4 小时）           |
| 指纹 ID    | 开启后验证交易密码时可用指纹代替                   |
| 结单提醒     | 设置结单推送提醒方式                         |
| 多重因素验证   | 账户二次验证设置                           |
| 资产隐私设置   | 控制他人是否可查看你的资产信息                    |
| 自动还款     | 开启后账户存在欠款时自动用其他币种余额兑换还款，避免产生额外融资利息 |
| 联系方式     | 设置接收证券类信息的手机号/邮箱，可与登录账号不同          |

## 交易设置

**功能设置：**

| 设置项    | 说明                  |
|--------|---------------------|
| 快捷下单   | 配置快捷交易模式            |
| 下单偏好设置 | 配置默认下单参数（数量、有效期等）   |
| 订单查询设置 | 订单列表的展示偏好           |
| 下单二次确认 | 开启后下单前需额外确认（建议保持开启） |
| 委托后面板  | 下单后是否自动展示委托详情面板     |

**交易提示开关：**

| 提示项          | 说明                 |
|--------------|--------------------|
| 交易提醒音效       | 委托成功/成交时的提示音       |
| 高买低卖提醒       | 下单价格高于买入价或低于卖出价时提醒 |
| 平仓冲突提醒       | 下单可能影响持仓平仓时提醒      |
| 市价单交易提醒      | 发出市价单前提醒           |
| 最小变动单位提醒     | 价格不符合最小变动单位时提醒     |
| 卖空提醒         | 卖空操作前提醒            |
| 快捷改单提示       | 使用快捷改单时提示          |
| 长期有效订单风险提示   | 设置长期有效订单时的风险提示     |
| OTC 订单延迟行情提醒 | OTC 订单使用延迟行情时提醒    |
| 末日期权交易提示     | 交易到期日期权时提示         |
| 非常用货币交易提示    | 使用非常用货币交易时提示       |

## 消息设置

| 设置项   | 说明                |
|-------|-------------------|
| 通知开关  | 跳转系统通知权限页面开启或关闭推送 |
| 资产与交易 | 订单成交、资产变动等交易类通知   |
| 资讯与社区 | 行情资讯、社区互动等运营活动通知  |

## 资讯与直播

| 设置项      | 说明                     |
|----------|------------------------|
| 自选资讯入口设置 | 选择「关注」Tab 中资讯入口默认展示的内容 |
| 直播悬浮窗播放  | 退出直播页后以悬浮窗继续播放         |

## 隐私

| 设置项      | 说明                       |
|----------|--------------------------|
| 个性化推荐    | 开启后展示基于行为分析的个性化内容推荐      |
| 公开我关注的股票 | 控制他人是否能在你的个人主页看到你关注的股票   |
| 公开我关注的股单 | 控制他人是否能在你的个人主页看到你关注的股单   |
| 黑名单      | 管理黑名单用户（被拉黑用户无法关注你或与你互动） |
| 隐私相关文档   | 隐私政策、个人信息收集情况、第三方共享清单等   |

## 关于

| 项目        | 说明              |
|-----------|-----------------|
| 当前版本      | 显示当前安装的 App 版本号 |
| 检查更新      | 检测是否有新版本可用      |
| 版本介绍      | 查看当前版本更新内容      |
| 关于长桥      | 公司介绍页面          |
| WHO'S LEO | 长桥吉祥物 LEO 介绍    |
| 服务协议      | 服务协议全文          |
| 市场行情服务条款  | 行情服务相关条款        |

## 其他入口

- **切换账号** — 退出当前账号并切换到其他账号
- **退出登录** — 退出当前账号登录状态
