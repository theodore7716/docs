# 银行卡绑定与审核

绑卡规则、信息填写、认证要求及常见问题解决。支持的银行列表详见[支持银行列表](/deposit/setup/supported-banks)。

## 绑卡规则

按监管要求，银行卡绑定数量受两层限制：

- **最多两家银行**：以 SWIFT CODE 区分，包含历史变动记录
- **每家银行最多两个账户**：同一银行下不同账号视为不同账户

因此账户绑定上限为 **2 家银行 × 2 个账户 = 4 张卡**。

例如：工银绑定 2 个账户 + 中银绑定 2 个账户，即达到上限，无法再绑定任何其他银行卡。

如需绑定新卡，需先在 App 内删除不用的银行卡或账户。

长桥支持香港卡、非香港卡入金。非香港地区银行卡目前仅支持 FATF 成员国家/地区的跨境银行转账，不支持大陆内地银行卡入金。请勿使用非 FATF 成员国家/地区银行卡直接转账至长桥账户，入金会被退回且退回手续费需自行承担。

## 添加银行卡

添加入口：
- 长桥 App - 资产 - 银行卡 - 添加银行卡
- 存入资金 - 选择入金币种 - 选择银行卡页面底部「管理银行卡」
- 提取资金 - 选择「提取至右侧更多」- 底部「管理我的银行卡」

### 填写银行卡信息

- 银行账户地区：选择银行卡所属地区（香港、美国、新加坡等）
- 卡所属银行：选择对应银行名称。香港地区银行卡需确认银行代码，如中国银行存在多个银行代码（012、014、338、070 等），需咨询开户行工作人员，或查看银行开户单账户号码前的分行号；选错分行会影响出入金
- 银行账户姓名：下拉选择，姓名排序需与银行端一致（如 ZHANG SAN 或 SAN ZHANG），选错会导致入金失败
- 银行账号：填写银行卡账户信息。部分银行存在综合账户卡号，不同币种对应卡号不同，入金前需提前确认
- 账户类型：根据银行卡号在银行端对应的账户类型选择
  - 港元账户：出入金仅支持港元
  - 美元账户：出入金仅支持美元
  - 综合账户：出入金支持美元和港元。绑定卡后如在入金页面看不到该卡，需检查账户类型与入金币种是否匹配，可在「资产 → 银行卡」中修改
- FPS 账号：选填，填写后提交出入金申请时可更快审核确认
- 银行卡证明：需同时包含客户姓名和银行账号的银行资料，如银行卡、银行结单、开户证明等。认证资料错误会影响出金流程，可能导致出金失败并产生银行费用

## 银行卡授权状态

银行卡可能处于以下状态：eDDA 已授权、银证已授权、审核中。

处于这些状态下，无法直接删除银行卡，也无法直接修改银行账户姓名和银行卡号信息；如需修改，请联系客服进行人工处理。

## 银行卡认证（出金前）

首次出金前，银行卡需完成认证。在提取资金页面如提示银行卡未认证，需点击「立即认证」上传银行卡认证资料。

认证资料必须同时包含客户姓名和银行账号，可选择以下任意一种：
- 银行月结账单
- 银行卡正面（需包含姓名，部分银行卡正面无姓名的需提供带姓名的实体卡截图）
- 开户成功的回执

支持最多上传 3 张图，提交后预计 1-3 个工作日处理完毕。请确保认证资料与长桥账户同名。

## 开户中绑卡

开户过程中只允许绑定一张银行卡，若有多次绑卡行为，默认为编辑更新银行卡信息。

## AML 风控审核

绑卡数量超出监管限制会触发风控审核。常见拒绝原因：

- 已绑定 2 家不同银行（以 SWIFT CODE 区分），无法再绑定第三家银行的卡
- 同一银行已绑定 2 个账户，无法在该银行绑定第三个账户
- 系统提示重复：该卡已存在，无需重复添加，请检查「资产 → 银行卡」中是否已有该卡及账户类型是否匹配

如有在途入金需要绑卡操作等特殊情况，请联系客服协助处理。

## 删除银行卡

删除路径：长桥 App - 资产 - 银行卡 - 选择具体银行卡 - 删除。

如银行卡已完成 eDDA 授权、银证授权、入金已见证等，不支持直接删除，必须先解除授权后才能删除。

如入金申请被驳回并提示银行卡信息填写错误，驳回并非退款——在「存入资金 → 入金记录」点击该记录，查看错误字段，点击「立即修改」更正银行卡信息后重新提交。

## 姓名多音字问题

如因姓名多音字导致银行卡绑定信息不一致，请联系客服并提供可证明姓名正确读音的证件（如港澳通行证、护照、银行结单等），协助修改为正确姓名。

## 信用卡限制

长桥不支持信用卡作为入金方式，也不支持绑定信用卡：

- 不支持使用信用卡进行开户入金验证
- 如通过信用卡汇款入金，资金到账后将被原路退款，退款处理约 14 个工作日，退款手续费由客户承担
