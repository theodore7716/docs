# FPS 转数快入金

FPS 转数快入金的收款信息、到账时间、手续费及操作步骤。

| 项目 | 说明 |
|------|------|
| 支持币种 | 港元（HKD） |
| 预计到账时间 | 港股交易日（周一 09:00 — 周五 23:59）2 小时内；其他时段下一港股交易日 09:40 前 |
| 手续费 | 长桥免费；大部分银行免费，个别机构可能收费，请向银行确认 |
| 转账限额 | 长桥无限额，请留意银行端限额 |

## 收款信息

| FPS ID | 收款银行 | 收款人名称 |
|--------|---------|-----------|
| 169152691 | 中国工商银行（亚洲） | Long Bridge HK Limited |
| 122366339 | 星展银行（香港）有限公司 | Long Bridge HK Limited |

## 操作步骤

1. 打开长桥 App，进入**资产 → 存入资金**，选择港元，选择银行卡，点击「FPS 转数快」，复制 FPS ID

   ![复制 FPS ID](https://pub.pbkrs.com/uploads/2026/d5973dfa459361c96bf6703d6a0a1998)

2. 登录网上银行或储值支付工具，通过 FPS 转数快功能向该 FPS ID 发起转账
3. 转账完成后**立即**返回长桥 App，进入**资产 → 存入资金 → FPS 转数快**，上传汇款转账凭证

   ![上传汇款凭证](https://pub.pbkrs.com/uploads/2026/b29b87921e8eceab48520d8a275ea60d)

> 凭证必须在转账后立即上传，否则影响入金进度。

## 同名账户要求与处理时限

- 转账银行账户名必须与长桥证券账户名**同名**，不可使用他人账户，否则退款费用由客户自行承担
- FPS 资金即时转入收款账户，但需经人工审批后才存入证券账户
- 银行通知「已汇出」不等于长桥已收款，资金到达后需结算与审批
- 香港节假日不处理汇款业务，请预留时间
- 不接受直接存入现金
