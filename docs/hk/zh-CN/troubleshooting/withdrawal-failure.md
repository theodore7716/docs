# 出金失败怎么办

出金申请被驳回或长时间未到账，按以下步骤逐项排查。

## 第一步：查看驳回原因

进入长桥 App → 资产 → 提取资金 → 出金记录，找到该笔申请：

- **状态「已驳回」**：点击记录查看驳回原因，参照下方表格处理
- **状态「处理中」**：仍在正常处理流程内，参照「到账时间」等待
- **状态「已完成」但银行未到账**：跳至第三步

### 常见驳回原因及处理方式

| 驳回原因 | 处理方式 |
|---------|---------|
| 银行账户名与证券账户名不一致 | 只能出金至同名银行账户，更换本人同名账户重新申请 |
| 银行卡未绑定或已失效 | 在 App → 银行卡管理重新绑定，再提交出金 |
| 可提金额不足 | 检查资金是否已结算（见第二步）；或账户存在欠款影响可提金额 |
| 超出单笔或日限额 | 分多次提交，或申请大额出金审核 |
| 账户存在限制 | 联系客服确认账户状态 |
| 出金至第三方账户 | 不支持出金至非本人账户、联名账户或支付平台 |

## 第二步：确认资金已结算

交易所得资金需等待结算后才可出金：

| 市场 | 结算周期 |
|------|---------|
| 港股 | T+2（交易日起算第 2 个工作日） |
| 美股 | T+1（交易日起算第 1 个工作日） |
| 新加坡股票 | T+2 |

如果当日刚完成交易，资金尚在结算中，需等结算完成后才可出金。  
App → 资产 → 现金页面可查看「可出金余额」（已结算）与「总资产」的差异。

## 第三步：确认到账时间

已审批通过的出金申请，按以下时间预期到账：

**香港账户 — FPS 转数快（港元）**

| 提交时间 | 预计到账 |
|---------|---------|
| 交易日 09:00–15:30 | 最快 2 小时内 |
| 其他时段 | 下一交易日 10:00 前 |

**香港账户 — 网银转账（港元 / 美元）**

交易日 15:30 前提交 → 当天出金；15:30 后 → 次一交易日出金

**电汇（美元）**：2–5 个银行工作日

**新加坡账户**：PayNow 实时到账；DDA 实时到账

如已超出上述时间仍未收到，继续下一步。

## 第四步：确认收款银行账号

若收款账号填写有误，资金可能转入错误账户。联系银行查询该笔汇款的实际收款方。

如确认银行账号有误：
1. 立即联系长桥客服说明情况
2. 提供汇款流水号和转账金额
3. 长桥将协助联系银行追回或重新处理

## 仍未解决

准备以下信息联系客服：

- 出金申请的记录截图（含申请时间、金额、状态）
- 银行流水截图（如已从银行账户扣款）
- 收款银行账号信息

**在线客服**：长桥 App → 我的 → 帮助与客服 → 在线客服
