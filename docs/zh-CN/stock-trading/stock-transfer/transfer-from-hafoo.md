# 从哈富证券转仓

从哈富证券转入股票分两步：先在长桥提交转入申请，再联系哈富证券获取表格并发邮件申请转出。须使用**哈富开户时绑定的邮箱**发送邮件。

> **注意**：哈富证券美股 DTC 代码为 **0158**，与通用代码（0534）不同，填表时请注意区分。
>
> 转入长桥不收费；转出费用由哈富证券收取。

## 第一步：在长桥提交转入申请

1. 打开**长桥 App** → **资产** → **存入股票** → **提交转入申请**；或进入**资产 → 全部功能 → 转入股票**

   ![存入股票入口](https://pub.pbkrs.com/uploads/2025/3869bd78857bb1dfe21a167da9659952)

   ![全部功能 → 转入股票](https://pub.pbkrs.com/uploads/2025/3776b6a7f3ccc2b9d6a9c8c70e402ecf)

2. 填写转出券商信息：

   | 字段 | 填写内容 |
   |------|---------|
   | 转出券商 | 其他证券 |
   | 券商英文名称 | Hafoo Securities Limited |
   | CCASS 代码（港股） | B01885 |
   | DTC 代码（美股） | **0158**（注意：哈富专用，非通用 0534） |
   | 联络人姓名 | Settlement Department |
   | 电话 | 852-39596115 |
   | Email | hk@eastmoney.com |
   | 账户姓名 | 您在哈富证券的账户姓名（须与长桥账户姓名一致） |
   | 账户号码 | 您在哈富证券的账户号码 |

3. 填写转入股票信息（股票代码、数量），确认后提交申请

   > 长桥支持填写每股成本价（选填）。未填写时按转仓成功当日收盘价计算；填写后无法修改，如有疑问请联系客服。

## 第二步：通知哈富证券转出股票

> 转仓表格需向哈富证券客服索取，**填表前请联系哈富客服确认填写信息**。

1. 联系哈富证券客服获取转仓申请表格，按以下长桥接收方信息填写

   | 字段 | 港股 | 美股 |
   |------|------|------|
   | 接收券商 | 长桥证券（香港）有限公司 Long Bridge HK Limited | Long Bridge HK Limited |
   | CCASS / DTC 代码 | B02195 | DTC 0534 |
   | 联系人 | 交收部 | Settlement Team |
   | 联系人电话 | (+852) 3585 8944 / (+852) 3585 8915 | (+852) 3585 8944 / (+852) 3585 8915 |
   | 联系人邮箱 | settlement@longbridge.hk | settlement@longbridge.hk |

   **港股表格填写示例：**

   ![港股表格填写示例](https://pub.pbkrs.com/uploads/2025/42709c8bc700dec9b2fc1e57c2fc2703)

   **美股表格填写示例：**

   ![美股表格填写示例](https://pub.pbkrs.com/uploads/2025/fcb2ca8d7e34985c2e84cee9c3591a7d)

2. 表格填写完毕后拍照，用**哈富开户绑定邮箱**发邮件至 hk@eastmoney.com

   **主题**：申请转出股票

   **正文**（参考）：
   > 您好，以下是我的股票转仓表格（见附件），请查收并尽快安排转出股票。我已通知长桥证券接收，如有疑问，请电话联系：（您的手机号码）

   **附件**：转仓表格照片

完成后耐心等待，哈富转出后股票将在 **1–2 个工作日**存入长桥账户。
