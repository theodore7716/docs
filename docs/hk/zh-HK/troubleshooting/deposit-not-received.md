# 入金未到账怎么办

转账已在银行端完成，但长桥账户余额未更新时，请按以下步骤排查。

## 第一步：确认是否在预期到账时间内

不同入金方式的处理时间不同，先对照下表确认是否已超时：

| 入金方式 | 预期到账时间 |
|---------|------------|
| eDDA 快捷入金（港股交易日） | 5 分钟内 |
| eDDA（非交易日或节假日） | 下一个港股交易日 09:40 前 |
| FPS 转数快（港股交易日） | 2 小时内 |
| FPS 转数快（其他时段） | 下一个港股交易日 09:40 前 |
| 网银同行转账（HK） | 约 2 小时 |
| 网银跨行转账（HK） | 1–3 个工作日 |
| 海外电汇（USD） | 2–5 个交易日 |
| DDA（新加坡） | 实时到账 |
| PayNow（新加坡） | 实时到账 |

**如果仍在预期时间范围内**：继续等待，香港节假日期间顺延。

**如果已超出预期时间**：继续下一步。

## 第二步：确认银行端转账已成功

打开银行 App 或网银，查看该笔转账的状态：

- 如果显示「交易失败」或「退款」：资金已退回银行账户，需重新发起入金
- 如果显示「处理中」或「已扣款但未到账」：继续下一步
- 如果显示「成功」但长桥未到账：继续下一步

## 第三步：检查入金申请信息是否有误

进入长桥 App → 资产 → 存入资金 → 入金记录，找到该笔申请：

- 如果状态为**「审核中」**：等待审批，通常 2 小时内完成（节假日顺延）
- 如果状态为**「已驳回」**：点击记录查看驳回原因，按提示修改后重新提交
- 如果**找不到该记录**：可能未完成入金申请，仅完成了银行转账但未在 App 内提交凭证（网银入金需上传汇款截图）

### 常见驳回原因

| 驳回原因 | 处理方式 |
|---------|---------|
| 账户名不一致 | 确认银行账户名与长桥账户名完全一致，使用本人同名账户重新转账 |
| 汇款备注缺失 | 网银/电汇须填写证券账户号+姓名；重新转账并填写备注 |
| 转账金额与凭证不符 | 重新上传正确的汇款凭证 |
| 非 FATF 成员地区 | 境外账户须来自 FATF 成员国家/地区，不接受中国大陆内地银行卡 |

- 长桥入金记录截图（如有）
