# App 登录失败怎么办

无法登录长桥 App 时，根据具体表现定位原因并处理。

## 收不到验证码

使用手机号 + OTP 方式登录时收不到验证码，请参阅 [收不到短信验证码](/troubleshooting/sms-code) 排查。

## 提示密码错误

使用手机号 + 密码、Login ID 或邮箱方式登录时提示密码错误：

1. 点击登录页的「Forgot Password」重置密码
2. 系统会向绑定手机号或邮箱发送验证码
3. 完成验证后设置新密码

如果绑定的手机号已不再使用，无法接收验证码，请联系客服进行人工身份核验。

## 提示账户异常或已冻结

登录后提示账户异常、被冻结或被限制：

1. 记录页面上显示的具体提示文字
2. 点击页面上的「联系客服」或前往 App → 我的 → 帮助与客服
3. 提供账户手机号和异常提示内容，由客服核查原因

常见原因包括：多次输错密码触发安全锁定、账户被风控限制、合规审查等。

## 第三方登录无法匹配账户

使用支付宝、Google 或 Facebook 登录后提示账户不存在或创建了新账号：

- 第三方登录以授权账号关联的手机号或邮箱匹配长桥账户
- 如果当前第三方账号关联的号码与注册时使用的手机号不同，系统无法匹配，会创建新账号
- 切换回手机号 + OTP 方式登录原账户，然后在「我的 → 账户安全」绑定第三方账号

## App 无法打开或加载异常

如果 App 能打开但登录页无法加载，或点击登录没有响应：

1. 检查网络连接，切换到其他 Wi-Fi 或移动数据重试
2. 如使用 VPN 或代理，关闭后重试
3. 强制关闭 App 后重新打开
4. 如问题持续，参阅 [APP 网络连接排查](/troubleshooting/network-connection)

## 提示「该账户在其他设备登录」

长桥支持多设备同时登录，此提示通常表示账户安全策略触发，如怀疑账户被盗用：

1. 立即修改密码：登录后前往「我的 → 账户安全 → 修改密码」
2. 在「账户安全 → 登录设备管理」下线可疑设备
3. 如无法登录，联系客服冻结账户

## 仍无法解决

点击登录页右上角「Get Help」直接联系客服，或：

- **在线客服**：长桥 App → 我的 → 帮助与客服 → 在线客服
- **客服热线**：4000712688
