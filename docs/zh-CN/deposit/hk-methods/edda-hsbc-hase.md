# 恒生 eDDA

恒生银行的 eDDA 授权通过网上银行的「P.P 支付平台」操作，授权完成后在长桥 App 发起入金，无需每次上传凭证。

> eDDA 入金的到账时间、手续费及通用故障排查，见 [eDDA 入金](/deposit/hk-methods/edda)。

## 第一步：网上银行授权

1. 登入**恒生银行网上银行** → **P.P 支付平台** → **直接付款授权书** → **建立新指标**

   ![P.P 支付平台入口](https://pub.pbkrs.com/uploads/2025/977b09c0a872b00a2cc3e3d877f378bb)

2. 在**类别**中选择 **OTHERS**，收款人选择**其他指定收款人**，然后完成保安编码校验

   > **注意**：收款人必须选择「其他指定收款人」，否则无法继续提交。

   ![选择收款人类型](https://pub.pbkrs.com/uploads/2025/37b505b9c9448763b08c9971185695a0)

3. 建立新指示，填写以下信息：

   | 字段 | 填写内容 |
   |------|---------|
   | 银行编号及名称 | 024 - 恒生银行有限公司 |
   | 户口号码 | 752027854001 |
   | 收款人名称 | Long Bridge HK Limited |
   | 债务人参考 | 您的长桥账号（即长桥 App 授权指引页面中的「付款人编号」） |
   | 最高金额 | 按需填写，建议不超过港币十万元；超过十万元需另外下载并邮寄授权书表格 |
   | 到期日 | 按需设置，建议不设到期日 |

   ![填写授权信息](https://pub.pbkrs.com/uploads/2025/14f8383a11ebc25fb1bc2ae2c5535bad)

4. 按网银指引获取**保安编码**，输入**交易保安编码**，点击**同意**

   ![输入保安编码](https://pub.pbkrs.com/uploads/2025/4cfe31c9c0c5de97d6eb926989ae2745)

5. 核对信息无误，点击**确认**，完成授权申请

## 第二步：长桥 App 发起入金

提交授权后，等待银行和长桥审批（预计 1–2 个银行工作日生效）。授权成功以银行端通知为准。

授权生效后：长桥 App → **资产** → **存入资金** → **存入港元**，选择恒生银行卡，点击「立即入金」。
