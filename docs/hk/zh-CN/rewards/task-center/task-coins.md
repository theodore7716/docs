# 任务币

任务币是 App 内的虚拟积分货币，用于激励用户参与平台各项功能和活动。

核心属性： - 不可充值购买，只能通过完成任务、签到、活动等方式获得 - 不可提现，不能兑换为现金提取 - 有有效期，每笔任务币有独立的到期时间，过期后自动失效，不可找回或延期 - 仅限平台内使用，只能在兑换商城消费或在猜涨跌游戏中消耗

## 生日特权

自 2025 年 8 月 1 日起，生日当天的 23888 任务币奖励不再发放。调整为商城折扣兑换，可在生日当月于商城享受指定礼品的折扣兑换。

## 获取方式

| 获取途径 | 说明 |
| --- | --- |
| 完成任务中心各类任务 | 包括每日/每周/每月/新手/成长/限时共 6 种任务类型，完成条件后系统自动发放 |
| 完成任务包阶梯奖励 | 任务包内完成一定数量的子任务后，按阶梯发放奖励 |
| 每日签到 | 每天签到可获得基础任务币，连续/累计签到可获得额外奖励 |
| 猜涨跌游戏预测正确 | 预测股票涨跌方向或区间正确后获得任务币奖励 |
| 参与 App 内特定活动 | 平台不定期举办的运营活动 |

## 消费方式

| 消费途径 | 说明 |
| --- | --- |
| 兑换商城兑换商品 | 可兑换卡券、实物周边、在线课程、现金、股票份额、App 道具等 |
| 猜涨跌游戏下注 | 参与猜涨跌需消耗任务币 |

## 到账时间

任务币到账时间以具体活动说明为准。如无特殊说明，默认在条件达成后 **5 个工作日**内到账。

## 有效期规则

- 任务币默认有效期为 **365 天**，部分任务/活动可能配置不同有效期，以发放说明为准

- 每笔任务币有各自独立的到期时间，不同来源、不同时间获得的任务币到期时间可能不同

- 到期后自动从可用余额中扣除，不可找回、不可延期、不可申诉恢复

- 在任务币流水/账户页面可查看每笔任务币的到期时间，临近到期会在积分界面显示到期时间

- 建议优先使用即将到期的任务币，定期检查余额和到期情况

- 累积任务币需在同一账号下，不同账号下的任务币不合并

## 反作弊条款

任务币一经使用无法返还，不可转让，不可折算现金或给予非任务币项目的给付。如发现违规行为（包括但不限于违反规则、恶意刷奖、用程式刷奖、伪造或虚报资料、使用科技手段获取不当利益等），平台保留收回相应任务币的权利。

## 任务币获取与兑换答疑

**任务币是什么？和现金有什么区别？**

任务币是 App 内的虚拟积分，不是现金。任务币通过完成任务、签到、活动等方式获得，只能在 App 内的兑换商城使用或在猜涨跌游戏中使用。任务币不可充值购买，也不可提现为现金。

**任务币怎么获得？有哪些途径？**

任务币的获取途径包括：完成任务中心的各类任务（每日/每周/每月/新手/成长/限时）、完成任务包的阶梯奖励、每日签到（连续签到还有额外奖励）、猜涨跌游戏预测正确、参与 App 内特定运营活动、系统补发（运营补偿）。

**任务币可以充值购买吗？**

不可以。任务币只能通过完成任务、签到、活动等方式获得，不支持任何形式的充值购买。

**任务币可以提现吗？**

不可以。任务币是平台内虚拟积分，不支持提现为现金或转账。只能在 App 内兑换商城使用或参与猜涨跌游戏。

**任务币会过期吗？过期了怎么办？**

会过期。每笔任务币有独立的有效期，到期后自动失效。过期的任务币无法找回、无法延期、无法申诉恢复。建议定期查看任务币流水中的到期时间。

**怎么查看任务币余额？**

可在以下位置查看：任务中心页面、兑换商城页面顶部。

**怎么查看任务币流水记录？**

进入任务中心，找到”任务币记录”入口，可查看每一笔任务币的获取和消耗记录，包括来源/用途、数量、时间和到期时间。

**为什么我的任务币余额减少了？**

任务币余额减少通常有以下原因： 1. 过期失效，部分任务币到了有效期自动失效，可在流水记录中确认 2. 兑换消费，在兑换商城兑换了商品，任务币被扣除 3. 猜涨跌消耗，参与猜涨跌游戏消耗了任务币

可通过任务币流水记录查看具体的变动明细。若对扣减有疑问，请联系客服并提供账号信息。

**任务币可以转给别人吗？**

不可以。任务币绑定个人账户，不支持转赠、转账或在用户之间流转。

**不同来源的任务币过期时间一样吗？**

不一样。每笔任务币有各自独立的到期时间，不同来源（如签到、完成任务、活动奖励等）获得的任务币到期时间可能不同。请在任务币流水中查看每笔的具体到期时间。

**任务币余额和兑换商城显示的不一致怎么办？**

可能原因包括：页面数据缓存未刷新（尝试下拉刷新或重启 App）、刚完成任务或兑换导致数据同步有短暂延迟、部分任务币刚刚过期被自动扣除。如刷新后仍不一致，请截图两个页面的余额并联系客服处理。

**过期的任务币能找回吗？**

不能。过期的任务币自动失效后无法找回、无法延期、无法通过任何方式恢复。建议定期关注余额和到期时间，及时使用即将过期的任务币。

**任务币可以用来做什么？**

任务币目前有两种使用方式：在兑换商城兑换商品（卡券、实物周边、在线课程、现金、股票份额、App 道具等），以及在猜涨跌游戏中参与预测。

**怎样才能获得更多任务币？**

建议：坚持每日签到，争取连续签到额外奖励；关注任务中心的每日/每周/每月任务，周期性完成获取奖励；参与任务包挑战，解锁阶梯奖励；参与猜涨跌游戏，预测正确可获得奖励；关注 App 内不定期推出的限时活动。

**完成任务后任务币多久到账？**

条件达成后系统自动处理奖励，通常较快到账。若完成后超过 24 小时仍未到账，请联系客服并提供账号信息及任务名称。

## 任务币使用限制与过期规则

- 任务币过期失效和兑换消费均不可逆，请谨慎操作

- 每笔任务币的到期时间独立，不要将余额视为统一到期

- 所有时间判断以系统服务器时间为准，非用户设备本地时间

- 任务币是虚拟积分，不具备货币属性，不可充值、不可提现、不可转让
