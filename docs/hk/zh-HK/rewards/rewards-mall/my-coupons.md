# 我的卡券

「我的卡券」页面用于查看和管理账户内所有持有的卡券。入口在「我的」→「我的卡券」，或「我的」快捷功能栏。

![我的卡券全部](../images/coupons-all.png)

## 分类 Tab

页面顶部按卡券类型分 Tab 筛选：

**全部 / 行情卡 / 免佣卡 / 打新卡 / 现金卡 / 股票卡 / 抵扣卡**

## 卡券列表

每张卡券显示：

- **卡券名称**（如「HKD 0.1 股票现金卡」「港股终生免佣」「终生半价咖啡优惠券」）
- **适用范围标签**（如「股票、轮证、ETF · 仅支持长桥综合账户」「适用 US、HK 市场」）
- **有效期**（如「2025/09/29 - 永久」「2025/12/05 09:18 - 2026/01/04 09:18」）
- **状态标签**：使用中 / 待兑现 / 即将到期
- **操作按钮**：「立即使用」（需手动激活的卡券显示此按钮）

## 卡券状态说明

| 状态   | 含义               |
|------|------------------|
| 使用中  | 已激活，当前有效（如终生免佣）  |
| 待兑现  | 已发放，等待条件触发后兑现    |
| 即将到期 | 有效期不足 7 天，建议尽快使用 |

免佣卡、行情卡等自动激活类卡券发放后直接生效，无需手动操作，不显示「立即使用」按钮；股票卡、现金卡等需手动激活的卡券才显示该按钮。卡券有效期由发放规则决定，激活后不影响到期时间，请在截止日期前完成使用。使用后可在「全部」Tab 查看最新状态，并在「我的」→「奖励记录」查看对应奖励发放流水。

## 隐藏已失效卡券

页面底部「隐藏已使用/已失效卡券」开关，默认开启。关闭后可查看所有历史卡券记录。
