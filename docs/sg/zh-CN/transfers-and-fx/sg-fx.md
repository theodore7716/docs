# 货币兑换

证券账户内多种币种之间的相互兑换，方便资金在港股、美股、新加坡股票之间转换。

## 支持币种

新加坡元（SGD）、港元（HKD）、美元（USD）、离岸人民币（CNH）

| 来源币种 | 可兑换为 |
| --- | --- |
| 新加坡元（SGD） | 美元（USD）、港元（HKD） |
| 美元（USD） | 新加坡元（SGD）、港元（HKD） |
| 港元（HKD） | 新加坡元（SGD）、美元（USD） |
| 离岸人民币（CNH） | 新加坡元（SGD）（仅此一种） |

CNH 只能兑换为 SGD，不支持兑换 USD 或 HKD。

**注意**：港股公司行动（如股息）可能产生 CNH。长桥不支持离岸人民币直接交易，收到 CNH 后需自行通过货币兑换转换为 HKD 或 USD 后使用。

## 手续费

免费（不收取手续费）

## 兑换汇率

由银行供应商提供的实时汇率，以实际兑换时的汇率为准。页面显示参考汇率约每 60 秒自动更新，实际兑换金额以最终成交为准。频繁兑入兑出会存在汇率差。

## 处理时间

全年 7×24 小时。如系统长时间未响应导致兑换未完成，请联系客服协助处理。

## 操作流程

长桥 App → 资产 → 全部功能 → 货币兑换，查看当前参考汇率，输入金额后即可兑换。

## 精度与用途

兑换金额均四舍五入取整至小数点后两位。货币兑换功能仅用于交易结算，请勿用于货币投机，与交易无关的请求将被拒绝。提交兑换申请时需输入长桥交易密码（6 位数字）。
