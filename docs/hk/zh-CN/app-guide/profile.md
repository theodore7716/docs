# 个人中心

「我的」是 App 底部导航最右侧的个人中心，汇聚账户信息、权益入口、功能快捷方式和设置入口。

## 用户信息区

页面顶部展示：

- 头像、昵称、认证标签（如「长期价值投资者」）
- **登录 ID**（格式如 `32*****88`），可用于 Login ID 登录方式
- 关注数 / 关注者数 / 我的收藏数 / 浏览记录数，点击各项可跳转对应列表（点击「关注」查看你关注的用户，点击「关注者」查看关注你的用户）

## 专享特权卡片

蓝色卡片，显示：

- **累计已省**：通过免佣等权益节省的交易费用合计（以正常费率应缴金额减去实际支付金额计算，持有免佣权益时实际支付通常为 0）
- 权益快捷图标（港股免佣、美股免佣、行情、融资利率等），点击跳转对应权益详情
- **待解锁提示**（如「2 项待解锁」），点击查看可解锁条件

## 快捷功能栏

横向滚动的图标入口：

| 图标      | 功能         |
|---------|------------|
| [LB Café](/rewards/lb-cafe) | 长桥咖啡兑换入口   |
| 我的卡券    | 查看持有的所有卡券  |
| 我的费率    | 查看当前账户适用费率 |
| 奖励记录    | 历史奖励明细     |
| 行情商店    | 购买或兑换行情权限  |

## 功能入口区

| 功能     | 说明               |
|--------|------------------|
| 任务中心   | 每日/周/月任务，完成获得任务币 |
| 邀请中心   | 查看邀请码，邀请好友注册     |
| 活动中心   | 查看当前可参与的活动       |
| 机会洞察   | 发现交易热点           |
| 我的收藏   | 收藏的文章和帖子         |
| 浏览记录   | 近期查看的标的和内容       |
| 产品建议反馈 | 提交功能建议           |
| 帮助与客服  | 联系在线客服、查看帮助文档    |
| 设置     | 账户安全、通知、语言等系统设置  |
