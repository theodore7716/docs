# 开户被拒怎么办

开户申请被驳回时，App 内会显示驳回原因。按原因对应处理后可重新提交。

## 常见驳回原因及处理方式

### 证件问题

| 原因 | 处理方式 |
| --- | --- |
| 证件照片不清晰或反光 | 在光线充足的环境下重新拍摄，确保四角完整、无遮挡 |
| 证件已过期 | 使用有效期内的证件，身份证需满足 6 个月有效期 |
| 证件与填写信息不符 | 检查姓名、证件号、出生日期是否与证件完全一致 |
| 不支持的证件类型 | 需提供身份证或护照，部分地区的驾驶证需人工审核 |

### 地区与身份限制

| 原因 | 处理方式 |
| --- | --- |
| 美国 / 加拿大证件或居民 | 长桥暂不支持美国和加拿大证件或居民开户 |
| 所在地区不在支持范围内 | 参考哪些国家和地区可以开户 |
| 大陆居民缺少海外生活证明 | 需补充新加坡或其他海外签证实体卡（新加坡账户要求） |

### 身份认证未完成

| 原因 | 处理方式 |
| --- | --- |
| 入金见证金额不足 | 香港账户入金见证需一次性入金 10,000 HKD 或 1,500 USD 以上 |
| 第三方见证材料未寄达 | 确认邮寄已成功，在 App 内点击「已寄出」确认 |
| 人脸识别失败 | 在光线充足环境下重新进行，确保面部无遮挡 |
| CA 验证未通过（大陆用户） | 重新进行人脸识别，或联系客服确认银联卡绑定状态 |



## 重新提交申请

修正问题后，在 App → 资产 → 开户申请 → 继续申请，从驳回的步骤重新提交，无需从头填写。

## 联系客服

如驳回原因不明确，或需要协助处理特殊情况，请联系客服：

**在线客服**：长桥App → 左上角头像 → 帮助与客服 → 在线客服
**香港邮箱**：service@longbridge.hk
**新加坡邮箱**：contact@longbridge.sg
