# 基金

资产 Tab 的基金页展示账户内的基金持仓情况，并提供余额通增值计划的入口和管理功能。

## 页面入口

资产页 → 全部功能 → 基金，或持仓列表中切换至「基金」标签。

## 持仓概览

| 字段   | 说明              |
|------|-----------------|
| 总市值  | 当前所有基金持仓折算后的总市值 |
| 持仓收益 | 当前浮动盈亏金额        |
| 累计收益 | 历史已实现与未实现收益合计   |

> 明细数据经四舍五入处理，与汇总数据可能存在细微偏差，请以结单数据为准。

## 快捷链接

- **交易记录** — 查看基金买入/卖出历史
- **基金盈亏** — 查看各只基金的盈亏详情
- **基金定投** — 设置或管理定期定额投资计划

## 余额通增值计划

余额通是长桥的闲钱增值服务，将账户闲置现金自动申购货币市场基金，实现资金不闲置。

### 主要特点

| 特点   | 说明                      |
|------|-------------------------|
| 自动申购 | 一次签约，闲置资金自动申购，无需手动操作    |
| 随时可用 | 支持随时手动赎回，最晚 T+1 日到账，赎回后可立即用于交易、出金或换汇；出金时系统自动赎回（1~2 个交易日），货币兑换或主动还款需手动赎回 |
| 低门槛  | 起购金额低，单账户无资金上限          |
| 低风险  | 主要投资流动性良好的货币市场基金        |

### 开通流程

1. 点击「开通余额通」
2. 阅读并同意相关协议（服务授权、基金销售协议、风险披露等）
3. 完成签约，系统自动开始申购

收益参考 7 日预期年化收益率，最终以实际到账金额为准，不保证最低收益。

> 详细规则请参考[余额通](/funds-and-wealth/余额通/香港余额通)。
