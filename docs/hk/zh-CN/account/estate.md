# 遗产处理

遗产继承所需文件清单及香港法院遗产承办处咨询方式。

## 办理流程

遗产承办人请将以下文件通过邮件发送至长桥香港官方邮箱（可委任香港律师办理）。

提交邮箱：service@longbridge.hk

## 所需文件清单

1. 香港法院的遗产继承文件（非香港客户也可至香港法院申请）
2. 已故用户的死亡证明书正本或核证副本
3. 已故用户身份证正本（已打孔）
4. 继承人的身份证明文件正本
5. 继承人与已故客户之间的关系证明文件正本，如结婚证明书、父母/子女的出生证明书

## 遗产承办处咨询

如需查询申请所需文件，可联系香港法院遗产承办处，电话：+852 2869 0869。
