# 撤销订单

已提交但尚未完全成交的订单可以撤销。全部成交或已过期的订单无法撤销。

## 撤销单笔订单

1. 进入「资产」→「订单」→「待成交」
2. 找到需要撤销的订单，点击进入订单详情
3. 点击「撤单」，确认撤销

撤销成功后，已占用的资金（买入委托）或股票（卖出委托）会立即释放回可用余额。

## 批量撤销

在「待成交」订单列表页右上角，点击「全部撤单」可一次性撤销所有未成交订单。

## 哪些订单可以撤销

| 订单类型 | 可撤销 | 说明 |
|---------|--------|------|
| 限价单（未成交） | ✓ | 随时可撤，交易时段内立即生效 |
| 限价单（部分成交） | ✓ | 撤销后已成交部分保留，未成交部分取消 |
| 市价单 | ✗ 或极短窗口 | 提交后几乎即时成交，通常无法撤销 |
| GTC / GTD 长期单 | ✓ | 可在任意交易日撤销 |
| 附加订单（止盈止损） | ✓ | 在触发前可撤销 |
| 到价反弹订单 | ✓ | 触发前可撤销 |
| 网格交易 | ✓ | 停止网格后，未成交的子订单自动撤销 |

GTC / GTD 订单在拆合股等公司行动发生时会被系统自动取消，详见[长期订单影响](/stock-trading/corporate-actions/impact-on-orders)。

## 不同时段撤单

**交易时段内**：撤单请求实时发送至交易所，通常秒级完成。

**非交易时段**：港股与美股订单可在非交易时段预挂，提交撤单后将在下一个交易时段开始时生效。

**港股开市竞价（09:00–09:20）**：此时段内不支持撤单，须等竞价结束后才可撤销未成交订单。

## 撤单后资金与持仓

- **买入委托撤销**：冻结的现金立即释放回账户可用余额
- **卖出委托撤销**：冻结的股票立即释放回可卖出股数
- 已成交部分不受影响，按原价格结算

## 撤单异常处理

**Q：撤单后显示「撤单中」持续很长时间怎么办？**  
港股交易时段内通常 30 秒内完成。如长时间未变更，可刷新订单列表。若仍异常，联系客服确认订单状态。

**Q：提交了市价单但想立即撤销，怎么办？**  
市价单会立即以当前最优价格成交，通常来不及撤销。建议改用限价单以便控制价格和撤单时机。
