# 出金

新加坡账户支持出金至新加坡本地银行卡、电汇至海外银行，或通过 Wise 出金。

操作路径：长桥 App → 资产 → 全部功能 → 提取资金，选择币种和收款银行卡。

支持币种：新加坡元（SGD）、美元（USD）、港元（HKD）

## 出金收费与到账时间

| 货币 | 提款金额 | 收款银行 | 预计到账时间 | 长桥手续费 | 预估银行收费 |
| --- | --- | --- | --- | --- | --- |
| SGD | ≤ 200,000 | 新加坡本地银行 | 1 个工作日 | 免费 | 无 |
| SGD | ≤ 200,000 | 其他 | 1–3 个工作日 | 免费 | 无 |
| SGD | > 200,000 | DBS / POSB | 1–3 个工作日 | 免费 | 无 |
| SGD | > 200,000 | 其他 | 1–3 个工作日 | 免费（豁免） | 约 SGD 10 |
| USD / HKD | 任意金额 | DBS / POSB | 1–3 个工作日 | 免费 | 无 |
| USD / HKD | 任意金额 | 其他银行 | 1–3 个工作日 | 免费（豁免） | 约 USD 35–60 同等值 |
| USD / HKD | 任意金额 | Wise | 审核约 1 工作日 + 出金 1–3 工作日 | 免费 | 自行承担 |

## 不支持的出金方式

PayNow、DDA 不支持出金。

## 限制与注意事项

资金只能提取至与长桥账户同名的银行账户，不能提取到他人账户、第三方支付平台或联名银行账户。银行账户需完成认证后方可出金。

只能提取已结算的金额，且账户现金净余额需大于 0。汇款费用如有变更恕不另行通知，汇款银行、收款银行及中间银行产生的费用均由客户承担。
