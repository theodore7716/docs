# 入金方式

香港账户的各类入金方式、支持货币、到账时间、手续费及限制说明。

需要帮助选择？如何选择入金方式 根据资金紧急程度给出推荐。

## 持有香港银行卡

| 入金方式 | 预计到账时间 | 手续费 |
| --- | --- | --- |
| 银证转账 | 即时 | 免费 |
| eDDA | 即时 | 免费 |
| FPS 转数快 | 约 2 小时 | 免费 |
| 网银转账 | 1-2 个工作日 | 视银行而定 |
| ATM / 柜台 | 1–3 个工作日 | 视银行而定 |
| 支票 | 约 2 个工作日 | 视银行而定 |
| 电汇 | 2-5 个工作日 | 视银行而定 |

## 非香港银行卡（境外汇款）

- 电汇：通过境外银行电汇转账至长桥香港收款账户

## 入金限制说明

- 入金账户姓名须与长桥证券账户姓名一致；非本人或联名账户汇款将被拒绝，退款手续费由客户自行承担

- 不接受大陆银行卡直接入金，不接受现金存入

- 信用卡入金将被拒绝并退回，预计处理约 14 个工作日，退回过程中的银行费用由客户承担

- Wise 入金需上传账户证明（含客户姓名、Wise 账户号码及 Wise logo）

- 如有多笔入金汇款，请针对每笔资金分别通知长桥

- 因追缴保证金通知而入金时，请及时填写存款通知，避免因不能及时到账导致账户被平仓
