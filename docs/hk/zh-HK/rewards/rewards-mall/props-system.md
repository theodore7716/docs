# 道具系统

道具是 App 内的功能性虚拟物品，目前主要用于猜涨跌游戏中提升预测正确后的奖励倍数。用户可在兑换商城使用任务币兑换道具。

## 道具用途

- 在猜涨跌游戏中使用，提升预测正确后的奖励倍数（如 2x、3x）
- 不是所有猜涨跌活动都支持道具，以活动页面显示为准

## 获取方式

| 途径 | 说明 |
|------|------|
| 兑换商城 | 使用任务币兑换，在商城中选择"道具"类商品 |
| 活动赠送 | 部分活动可能赠送道具，以活动页面说明为准 |

## 使用规则

- 在猜涨跌下注时选择使用道具，无法事后追加
- 每次猜涨跌只能使用一个道具
- 使用道具需额外消耗一定数量的任务币（具体数量以道具说明为准）
- 道具使用后不可撤销
- 如预测错误，道具效果不生效，但道具已消耗，不会退还

## 道具获取与使用答疑

**道具是什么？有什么用？**

道具是 App 内的功能性虚拟物品，目前主要用于猜涨跌游戏。在猜涨跌中使用道具可以提升预测正确后的奖励倍数（如 2x、3x），让猜对时获得更多任务币奖励。

**道具怎么获得？**

目前主要通过兑换商城使用任务币兑换。进入兑换商城后，选择"道具"类商品即可兑换。部分活动可能赠送道具，具体以活动页面说明为准。

**道具在猜涨跌中怎么使用？**

在猜涨跌游戏下注时，如果当前活动支持道具，页面上会显示道具选择入口。选择要使用的道具后确认下注即可。道具需在下注时选择，下注完成后无法追加使用。

**使用道具需要额外费用吗？**

需要。使用道具时除了道具本身外，还需额外消耗一定数量的任务币。具体消耗数量以道具详情说明为准，在使用前页面会明确展示所需的任务币数量。

**道具的奖励倍数怎么计算？**

使用道具后，如果猜涨跌预测正确，获得的任务币奖励会按道具标注的倍数提升。例如使用 2x 倍数道具，预测正确后获得的奖励为原来的 2 倍。具体倍数以道具详情页标注为准。如果预测错误，道具效果不生效，不会获得任何额外奖励。

**如果猜错了，道具会退还吗？**

不会退还。道具在下注时即视为已使用，无论预测结果正确还是错误，道具都会被消耗。预测错误时道具效果不生效，但道具不会退还。请谨慎选择是否使用道具。

**为什么我在猜涨跌中找不到道具选项？**

可能原因包括：当前活动不支持道具（不是所有猜涨跌活动都支持使用道具，以活动页面显示为准）、没有可用道具（需先在兑换商城使用任务币兑换道具）、任务币不足（使用道具需额外消耗任务币，余额不足时可能无法使用）。如确认以上条件均满足但仍找不到道具选项，请联系客服并提供账号信息和所参与的活动名称。

**道具有有效期吗？**

道具是否有有效期，以道具详情页的说明为准。建议在兑换道具后及时查看道具详情了解使用期限，避免过期浪费。

**一次可以使用多个道具吗？**

不可以。每次参与猜涨跌下注时只能使用一个道具，无法叠加使用多个道具。

## 道具消耗规则与使用限制

- 道具一旦在下注中选择使用，即不可撤销，无论预测结果如何道具都会被消耗
- 道具绑定个人账户，不支持转赠或在用户之间流转
- 并非所有猜涨跌活动都支持道具，请以活动页面实际显示为准
- 使用道具需额外消耗任务币，使用前请确认任务币余额充足
