# 每日签到

每日签到是一项每天打卡领取任务币的功能。用户每天完成一次签到即可获得基础任务币奖励。坚持连续签到或累计签到达到一定天数，还可额外获得更丰厚的奖励。

## 如何签到

参与签到需已登录账号，且当日尚未签到。

操作步骤：

1. 进入任务中心或签到页面
2. 点击"签到"按钮
3. 系统显示签到成功并发放基础奖励任务币
4. 若满足连续签到或累计签到条件，同时显示并发放额外奖励任务币

签到成功后页面显示"已签到"状态，基础奖励任务币到账；若触发额外奖励，会同时显示额外奖励金额。

## 签到状态说明

| 状态 | 含义 |
|------|------|
| 可签到 | 当前时间可以签到，今日尚未打卡 |
| 已签到 | 今日签到已完成 |
| 未签到 | 该日未完成签到（历史日期） |

## 签到奖励计算答疑

**签到是什么？有什么用？**

每日签到是一个每天打卡领奖励的功能。完成签到可获得任务币，任务币可用于兑换礼品或参与平台活动。坚持签到还能解锁连续/累计签到的额外奖励。

**如何签到？**

进入任务中心找到签到入口，点击"签到"按钮即可完成。每天只需操作一次。

**一天可以签到几次？**

每天只能签到一次。重复点击不会额外获得奖励，也不会影响已有签到记录。

**签到奖励多久到账？**

签到成功后任务币通常即时到账。若超过 24 小时仍未到账，请联系客服并提供账号信息。

**连续签到奖励是什么？怎么获得？**

连续签到奖励是指连续打卡 N 天后额外获得的任务币。只要不中断签到、每天按时打卡，达到指定天数后系统会自动发放奖励。具体奖励天数和金额以当前活动页面展示为准。

**我忘记签到了，连续签到记录断了怎么办？**

漏签后连续签到记录将会中断，重新从第 1 天开始计算。目前不支持补签或恢复之前的连续记录，建议每天定时打卡。

**连续签到和累计签到有什么区别？**

连续签到要求每天不间断打卡，中间漏一天记录清零重算，奖励基于"连续打卡天数"。累计签到只统计历史上签到的总天数，不要求连续，漏签不影响已累计天数，奖励基于"累计总天数"。两种模式取决于当前活动配置，页面上会有明确说明。

**签到记录在哪里查看？**

在签到页面或任务中心可以查看历史签到记录，包括哪些天已签到、当前连续/累计天数等信息。

**签到周期什么时候重置？**

签到周期分为"每周重置"和"每月重置"两种模式，具体以当前活动配置为准。每周模式在每周一 0 点重置，每月模式在每月 1 日 0 点重置。页面上会显示当前周期的开始和结束时间。

**为什么我昨天签到了，今天显示还是"未签到"？**

"未签到"是今日的状态，表示今天还未打卡，与昨天是否签到无关。每天零点后状态会刷新，需要重新签到才能变为"已签到"。如果确认今天已签到但状态异常，请联系客服处理。

**签到获得的任务币有过期时间吗？**

有。签到获得的任务币有有效期，过期后自动失效且不可找回。请在任务币账户页面或兑换商城查看各笔任务币的到期时间，建议及时使用。

**为什么签到按钮是灰色/点不了？**

可能原因：今日已签到、活动尚未开始或已结束、网络异常。请确认当前签到状态，若状态显示"可签到"但仍无法操作，请刷新页面或联系客服。

**每周额外奖励是什么？怎么领取？**

部分活动配置了每周完成一定次数签到可获得额外任务币奖励，达到条件后系统自动发放，无需手动领取。具体规则以当前活动页面展示为准。

## 签到有效性与奖励限制

- 签到时间以系统时间为准，非用户本地设备时间，跨时区用户需注意
- 每日零点后签到记录刷新，当日首次签到即为有效签到
- 连续签到模式下，漏签一天即中断，不可补签
- 累计签到和连续签到额外奖励不同时存在，取决于活动配置
- 历史签到记录不可修改
- 签到行为可推进任务中心中与签到相关的任务进度（如"签到 N 次"任务）
