# 期权条款

当您在长桥进行期权交易时，即表示您确认以下事项：

1. 您已了解期权交易具有高度投机性，并包含高度风险。

2. 您确认具备承担期权合约交易相关风险的经济能力，且能够承受因此产生的任何损失，包括：
   - 多头看跌期权和多头看涨期权所支付权利金的全部损失
   - 空头卖权和空头买权头寸相关的保证金维持要求
   - 交易成本

3. 您在卖出看跌期权和看涨期权时，须遵守长桥证券的保证金维持要求，且不得超过任何交易所、市场或有管辖权的监管机构规定的头寸/行使限额。

4. 在进行第一笔期权交易之前，您须向长桥证券确认已阅读并完全理解最新的期权清算公司（OCC）披露文件《期权的特征和风险》（OCC 文件），可在以下地址获取：[https://www.theocc.com/Company-Information/Documents-and-Archives/Options-Disclosure-Document](https://www.theocc.com/Company-Information/Documents-and-Archives/Options-Disclosure-Document)

5. 您确认期权合约在特定期间内交易。

6. 波动性、流动性和系统故障可能使特定市场的执行变得极为困难。

7. 您已了解电子交易服务是电子系统，可能存在无法使用的情况。

8. 您已确认：
   - 如账户净值不足以满足保证金维持要求，长桥证券或清算公司可自行决定清算您的任何头寸
   - 如损失等于或超过账户申请中指定的投机资本金额，交易特权将被终止
   - 清算公司可自动行使到期时价内 0.01 或以上的看跌期权或看涨期权，但无义务这样做

9. 您已阅读并充分理解长桥平台期权合约交易的保证金维持要求。

10. 期权合约受 OCC 规则和法规、其他监管机构以及合约交易所的具体规则约束。

11. 购买期权合约须在购买时全额付款，即期权合约不得以保证金购买；您已完全了解买卖看涨期权和看跌期权的保证金维持要求，以及此类交易所涉及的风险。

12. 期权合约由 OCC 发行，您不得自行或与他人共同违反 OCC 或合约交易所规定的持仓和行权限制。

13. 如您在期权合约到期之前没有足够净值以满足买入或卖出标的证券所需的初始保证金要求，执行经纪商可自行决定清算标的证券头寸，从而导致来自期权合约的行权。长桥证券、执行经纪商和清算公司均无义务代表您买入或卖出标的证券。

14. 您已了解 OCC 将行权分配给清算公司，且确认在被分配时，您将被要求：
    - 股权期权：交付或接受所需数量的标的证券股份
    - 股票指数期权：以现金支付或接收结算价格

    您已了解，在 OCC 向清算公司进行初始转让日起的一天或多天后，您可能不会收到长桥证券或其清算公司的转让通知，且缺乏此类通知会给未担保资产带来特殊风险。您确认已阅读并理解 OCC 文件第 VIII 章和第 X 章所述的风险。

15. 如执行多头看跌合约导致标的产品被卖空，则您在下单时对该订单负责；您确认卖空只能在符合初始保证金和保证金维持要求的保证金账户中执行。在执行卖空前，长桥证券的清算公司须能够代表您借入相关标的（此处仅适用股票），以交付给买方。如已代表您借入证券，但贷方公司随后向清算公司发出召回通知，长桥证券的清算公司将尝试代表您重新借入证券。若证券无法重新借入，您确认长桥证券可能在不通知您的情况下，通过在公开市场上以当前市场价格买入股票来弥补您的空头头寸，由此产生的所有相关费用由您承担。空头股票的市场价值被视为您保证金账户中的借方项目。

16. 账户将按长桥平台公布的每日利率收取或获得利息；您确认这些利率如有更改，恕不另行通知。利率适用于您欠长桥证券的所有借方余额，以及清算公司根据与长桥证券协议向您提供的所有信贷。在计算信用利息时，账户中持有的利息信用和股息被视为自由信用余额的一部分。利息以一年 365 天为基础，按结算日余额计算。
