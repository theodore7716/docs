# 招行香港网银转账

通过招商银行香港手机银行或网上银行将资金转至长桥，转账完成后上传凭证即可。

> 网银转账的到账时间、手续费及通用注意事项，见 [网银转账入金](/deposit/hk-methods/online-banking-transfer)。

## 手机银行

1. 打开**招商银行 App** → 点击**首页左上角**「扫一扫」，扫描以下二维码进入**招行香港分行**界面

   ![扫码进入香港分行](https://pub.pbkrs.com/uploads/2026/868fdeffea9907d9b5772ad34755dcd2)

2. 进入**招行香港一卡通**主页面 → **转账** → **香港本地**，填写收款人信息，点击**下一步**，核对无误后点击**确认转账**，提示成功即汇款完毕

   ![转账 → 香港本地](https://pub.pbkrs.com/uploads/2026/bc65f60e88e91db4f5468e38c991747b)

   ![填写收款信息确认](https://pub.pbkrs.com/uploads/2026/9bb185a04955ed1c7da699fc884d3c63)

3. 立即截图保留凭证，返回**长桥 App** → **资产** → **存入资金** → **网银转账**，上传凭证

   ![上传汇款凭证](https://pub.pbkrs.com/uploads/2026/84d0ba3afc1de5440586c174f6be6292)

## 网上银行

> **前置条件**：需先下载「个人银行香港专业版」PC 桌面应用。参见 [招行香港 eDDA 授权入金](/deposit/hk-methods/edda-cmb) 中的下载步骤。

1. 登录**招商银行香港分行个人网上银行**（http://hk.cmbchina.com），下载并通过**个人银行香港专业版**登录

   ![下载登录页面](https://pub.pbkrs.com/uploads/2026/8bd87ed6de96f7d8f8cb653c03fc3f59)

   ![个人银行香港专业版登录](https://pub.pbkrs.com/uploads/2026/53303e2b48e11c47a7286ea417b55a4a)

2. 选择**一卡通** → **转账汇款** → **香港本地转账**

   ![导航至香港本地转账](https://pub.pbkrs.com/uploads/2026/b903aeb5c79a48d52634f8b9e7d3b5a8)

3. 选择**其他银行**，填写收款人信息，核对无误后点击**确定**，提示成功即汇款完毕

   ![填写收款信息并确认](https://pub.pbkrs.com/uploads/2026/86bb69328d19dc2f6678115f2cec5919)

4. 立即截图保留凭证，返回**长桥 App** → **资产** → **存入资金** → **网银转账**，上传凭证

   ![上传汇款凭证](https://pub.pbkrs.com/uploads/2026/cd8021961ffab14492e7db46339c1417)

   > 凭证必须在汇款完成后立即上传，否则影响入金进度。
