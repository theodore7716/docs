# 恒生银行可疑账号澄清

长桥恒生银行收款账户被标示可疑的官方声明及替代方案。

## 背景

长桥证券（香港）App 内展示的恒生银行收款账户（账户号码：752027854001），被香港警务处「防骗视伏 App」标示为可疑，触发了防诈骗提示。

## 官方声明

该恒生银行账号属于长桥证券（香港）正式持有，用于为客户提供入金服务。账号本身没有任何安全问题，亦非诈骗相关账户。

长桥已就事件向相关部门申请覆核与澄清。

## 替代方案

如对该账户安全有所疑虑，可选择使用长桥提供的其他银行账号进行入金操作。所有收款账号信息请以长桥 App 内入金页面显示为准。
