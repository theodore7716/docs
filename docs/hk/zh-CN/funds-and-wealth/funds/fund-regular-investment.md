# 基金定投

基金定投（定期定额投资）是指按照预设的时间周期和固定金额，自动买入指定基金的投资方式。通过分散买入时点，平摊持仓成本，降低择时带来的风险。如果倾向于让闲余资金自动增长而非定期手动设置，也可参考[闲钱理财](/funds-and-wealth/idle-cash)。

## 入口

财富页快捷入口「基金定投」，或「更多」→「基金定投」。

也可在基金订单页切换至「基金定投」标签，查看已有定投计划。

## 设置定投计划

### 主要参数

| 参数 | 说明 |
|------|------|
| 基金 | 选择参与定投的基金产品 |
| 定投金额 | 每期自动扣款并买入的固定金额 |
| 定投周期 | 扣款和买入的时间频率，如每周、每月 |
| 定投日 | 在所选周期内的具体日期 |

### 非交易日顺延与余额不足规则

- 定投日如遇非交易日，顺延至下一个交易日执行
- 每期买入按当期 T 日净值确认，与普通买入规则一致（截止时间 09:10 香港时间）
- 账户可用余额不足时，当期定投将跳过执行，不自动补扣

## 交易规则

定投的每笔买入适用与普通基金相同的交易规则：

| 时间节点 | 说明 |
|----------|------|
| T 日 09:10 前触发 | 按 T 日净值确认买入 |
| 预估 T+3 日 | 买入份额确认到账 |
| 预估 T+3 日起 | 可查看持有收益 |

详细交易规则参见[基金交易规则](/funds-and-wealth/funds/fund-trading-rules)。

## 查看与管理定投计划

在「基金订单」页切换至「基金定投」标签，可查看当前所有定投计划的状态、执行记录及累计持有情况。

可对定投计划执行以下操作：

- **暂停**：停止后续自动扣款，已持有份额不受影响
- **恢复**：重新启用已暂停的定投计划
- **终止**：永久取消计划，不影响已购入份额的持有与赎回

## 费用

定投买入适用基金的正常申购费率，无额外定投手续费。具体费率以基金详情页说明为准，详见[基金收费](/funds-and-wealth/funds/fund-fees)。

## 风险提示

基金投资存在风险，定投不能保证盈利，也不能规避市场系统性风险。历史收益不代表未来表现，请根据自身风险承受能力谨慎决策。
