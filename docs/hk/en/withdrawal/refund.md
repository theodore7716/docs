# 资金退款

资金原路退回的触发场景、所需信息、退款时效及注意事项。

## 退款触发场景

以下情况可能需要资金原路退回：

- 开户申请被拒绝或主动终止开户
- 银行卡未绑定成功但已入金
- 使用非本人银行卡（第三方）入金
- 使用信用卡入金
- 使用不支持的银行卡（如内地银行卡、非 FATF 成员国银行卡）入金

## 退款所需信息

如需申请退款，请联系客服并提供以下信息：
- 汇款银行账户号
- 银行卡名称及银行代码
- 汇款人名称

## 退款时效

### 长桥香港

香港银行卡退款：预计 7 个工作日
非香港银行卡、第三方平台、信用卡退款：预计 2–4 周

### 长桥新加坡

以实际到账为准

退款需双方银行共同操作处理，时间会较长。

## 原路退款说明与跨境费用

- 退款为原路退回，退回至原汇款账户
- 跨境电汇退款可能产生银行手续费，相关费用由客户自行承担
- 如汇款显示为中转行账户而非本人汇款账户，可能需要提供银行账单协助核对
- 如银行账单中查询不到入金信息，说明长桥尚未收到资金，建议咨询汇款银行定位资金动向
