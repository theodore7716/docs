# eDDA 入金

eDDA 授权入金的开通步骤、支持银行及常见授权问题。

| 项目 | 说明 |
|------|------|
| 支持币种 | 港元（HKD） |
| 预计到账时间 | 港股交易日 5 分钟内；其他时段下一港股交易日 09:40 前（港股休市日顺延） |
| 手续费 | 免费；账户余额不足时部分银行（如汇丰）可能扣费，转账前请确保账户余额充足 |

## 操作步骤

### 第一步：完成 eDDA 授权

**方式一：在长桥 App 内发起（推荐）**

1. 打开长桥 App，进入**资产 → 存入资金 → 存入港元 → 香港银行卡**
2. 选择银行卡 → **eDDA 入金** → 申请授权（有 eDDA 标签的表示支持）

   ![选择银行卡发起 eDDA 授权](https://pub.pbkrs.com/uploads/2026/b4da71d014c075f888fb5dac35f6dc1a)

3. 确认银行授权信息（卡号、姓名），点击「确认授权」
   - 若已绑定过银行卡：确认卡号和姓名后点击「确认授权」（如有误可在「资产 → 银行卡」中修改）

   ![已绑定银行卡确认授权](https://pub.pbkrs.com/uploads/2026/27fdd31b93b0175ebccb5a077a241f36)

   - 若未绑定过银行卡：填写银行账户姓名、汇款银行账号、账户类型后点击「确认授权」

   ![未绑定时填写信息](https://pub.pbkrs.com/uploads/2026/d54842b5fd40c5534c2880b06ab49b03)

4. 等待授权结果，成功后收到推送消息和邮件通知

> 开户证件类型、姓名排序必须与银行开户时一致，否则授权失败。大部分银行约 5 分钟出结果，个别银行需 2–14 个工作日。

**方式二：在银行端发起**

花旗银行、东亚银行、招商银行需在银行网页端发起 eDDA 授权，授权成功后再在长桥 App 发起入金。在 App 内多次授权失败时也可尝试此方式。

各银行详细步骤：

- [中银香港 eDDA 授权入金](/deposit/hk-methods/edda-boc)
- [工银亚洲 eDDA 授权入金](/deposit/hk-methods/edda-icbc)
- [恒生银行 eDDA 授权入金](/deposit/hk-methods/edda-hsbc-hase)
- [招商银行（香港）eDDA 授权入金](/deposit/hk-methods/edda-cmb)
- [汇丰银行 eDDA 授权入金](/deposit/hk-methods/edda-hsbc)
- 其他银行：请在长桥 App 入金页面查看对应银行的操作指引

### 第二步：发起入金

授权成功后，在长桥 App 的 eDDA 入金页面点击「立即入金」，输入金额即可。无需上传入金凭证。

![eDDA 立即入金](https://pub.pbkrs.com/uploads/2026/b76af391629a9959f08e0f04200f4fa6)

## 解除授权与删除银行卡

- 长桥证券仅支持单方解除 eDDA 授权；银行侧拥有更高权限，可直接双向解除绑定
- 解除 eDDA 授权需在对应银行网银端操作，App 端不支持解除
- 需先解除 eDDA 授权，才能在 App 删除该银行卡
- 如在长桥侧解绑后再次绑定出现「重复授权」报错，通常是银行端未成功解除，需联系银行处理
- 如需关闭 eDDA，可联系客服在「证券后台 → 出入金方式 → eDDA → 修改状态」点击「已关闭」；同一个银行账号可能存在多条记录，关闭前请检查清楚

## 状态一直显示开通中

可能是银行代码选错。请直接「删除」该条授权记录，删除后需要审批（工作日预计 1 个小时左右），通过后重新发起授权即可。

## 常见授权失败原因

| 错误信息 | 原因 |
|---------|------|
| Debtor account name / ID not matched | 证件类型或姓名格式问题，字母需大写 |
| Invalid debtor account number | 账号错误、加了银行代码、或使用了分币种账号 |
| Invalid debtor ID number | 证件账号有误（字母未大写），多见于建行（009） |
| Mandate identification already exists | 银行端与长桥 App 端同时发起授权，多见于中银（012） |
| Debtor account not found | 账号有误，多见于汇丰、渣打 |
| Rejected by Other bank | 银行卡账户类型问题（如绑定了美元账户类型） |
| Invalid Payer ID | 银行卡证件信息与开户信息不一致（例如：用香港身份证开银行卡，但长桥开户时用的是大陆身份证），需更新 KYC 证件信息后重新授权 |

如排查完所有原因仍无法授权，可联系银行重置 eDDA 后再次尝试。华侨银行一般需 2 个工作日处理。

## 同名账户要求与出金限制

- 转账银行账户名必须与长桥证券账户名同名，不可使用他人账户转账
- 银行通知「已汇出」不代表长桥已收款，资金到达后需结算与审批
- 香港节假日期间不处理汇款业务，请预留时间
- eDDA 不支持出金
