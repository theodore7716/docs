# 抽奖活动

抽奖活动（Lottery Activity）是 LongBridge
平台推出的随机奖励活动。用户通过完成指定任务（如登录、交易、邀请好友等）获得抽奖机会（抽奖券），在活动规定的时间内使用抽奖机会参与抽奖，随机获得现金、股票、券或实物奖品。抽奖活动兼具趣味性和随机性，中奖概率以活动规则公示为准。

## 参与条件

- 持有有效的 LongBridge 账户，账户状态正常
- 已完成实名认证（KYC）或满足活动指定的基础条件
- 账户所属地区/持牌人在活动覆盖范围内
- 持有有效的抽奖机会（抽奖券）才能参与抽奖

## 获取抽奖机会的任务类型

- 每日登录：每天登录 APP 获得 1 次抽奖机会
- 完成交易：每完成指定金额或笔数的交易获得抽奖机会
- 邀请好友：每成功邀请并完成指定任务的好友获得若干次抽奖机会
- 入金任务：向账户入金达到指定金额获得抽奖机会
- 活动签到：连续签到积累抽奖机会
- 购买指定产品：购买基金、债券等特定产品获得抽奖机会

## 参与步骤

1. 在「活动中心」找到当期抽奖活动
2. 完成指定任务获取抽奖机会（抽奖券）
3. 进入抽奖页面，点击「立即抽奖」使用抽奖机会
4. 系统随机分配中奖结果
5. 中奖奖励自动进入发放流程
6. 在「活动中心 → 我的奖励」查看奖励发放状态
7. 未使用的抽奖机会通常在活动结束后失效

## 奖励类型

- 现金奖励：随机金额的现金，直接到账
- 股票奖励：指定股票若干股
- 交易券/现金券：抵扣手续费或消费使用
- 体验金：有期限和条件的模拟资金
- 实物奖品：电子产品、礼品卡等（通常需要填写收货地址）
- 谢谢参与：部分抽奖设有「未中奖」结果

## 发放时机

抽奖后中奖结果即时确定，奖励随即进入发放流程。现金/券类奖励通常在中奖后 1
个工作日内到账。实物奖品需要用户填写收货信息，邮寄周期另行说明。若中奖奖励处于待发放状态，以活动规则中的预期到账时间为准。

## 奖励状态说明

| 状态    | 含义             | 用户操作         |
|-------|----------------|--------------|
| 待发放   | 中奖已确认，奖励正在排队发放 | 无需操作，等待系统处理  |
| 发放中   | 系统正在将奖励发送至账户   | 等待，通常数小时内完成  |
| 已发放   | 奖励已成功到账        | 可在账户余额或持仓中查看 |
| 锁定中   | 奖励已到账但处于锁定期    | 等待锁定期结束自动解锁  |
| 已解锁可用 | 锁定期结束，可正常使用    | 正常使用奖励       |
| 发放失败  | 奖励发放失败         | 联系客服协助处理     |
| 已取消   | 奖励被取消          | 联系客服确认原因     |

## 抽奖机会

抽奖机会（抽奖券）是参与抽奖的唯一凭证，每次抽奖消耗 1 次，无论中奖与否。

### 获得方式

抽奖机会可通过以下方式获得：

- 完成活动页面指定任务（登录、入金、交易、邀请等）
- 邀请好友完成助力条件（双方同步获得）
- 完成任务中心任务后在兑换商城兑换
- 活动方直接发放（如运营补偿、专属活动）

### 查看与管理

- 当前持有的抽奖机会数量显示在活动页面顶部或抽奖按钮旁
- 每次获得新的抽奖机会，页面数量实时更新（偶有短暂延迟）
- 抽奖机会绑定到具体活动，不同活动的抽奖机会不能混用

### 使用规则

- 不可转让，只能本人账户使用
- 每次点击「立即抽奖」消耗 1 次机会，机会扣除后不可恢复（包括结果为「谢谢参与」的情况）
- 部分活动设有每日使用上限（如每日最多抽 3 次），超出上限当天无法继续，次日重置
- 活动结束后未使用的抽奖机会自动失效，不补偿、不退换

## 分享加码抽奖

特殊玩法：用户邀请好友通过专属链接参与助力，好友有效助力后双方的抽奖次数或奖励金额同步增加。助力需满足活动设定条件（如好友完成注册、登录等），仅点击链接不算有效助力。每位好友通常只能助力
1 次，同一好友重复助力不计入。

## 抽奖资格与结果答疑

**Q：我有抽奖机会，点了抽奖按钮，结果是「谢谢参与」，抽奖机会扣了吗？**
A：是的，每次点击抽奖按钮都会消耗 1 次抽奖机会，无论中奖与否（包括「谢谢参与」结果）都会扣除。抽奖机会一经使用不可恢复，请确认您希望使用后再点击抽奖。

**Q：我中奖了，奖励什么时候到账？**
A：中奖后奖励会自动进入发放流程。现金和券类奖励通常在中奖后 1 个工作日内到账，您可以在「活动中心 →
我的奖励」查看具体的奖励状态（待发放/发放中/已发放）。实物奖品需要您填写收货地址，邮寄时间另行告知。

**Q：抽奖机会有使用期限吗？活动结束后还能用吗？**
A：抽奖机会有使用期限，通常在活动结束时统一失效。活动结束后未使用的抽奖机会将自动作废，不做补偿。请在活动有效期内及时使用您获得的抽奖机会。具体有效期以活动详情页说明为准。

**Q：中奖后奖励显示「发放失败（Failed）」，怎么办？**
A：中奖奖励发放失败的常见原因：（1）账户 KYC 未完成；（2）账户状态异常；（3）系统技术问题。请先检查账户实名认证是否完成，账户是否正常，然后联系客服提供奖励详情，客服会协助重新触发发放或给出解决方案。

**Q：我每天登录应该得到抽奖机会，但今天的机会没有到账，是什么原因？**
A：每日登录奖励通常有特定的刷新时间（如香港时间每天 00:00 刷新）。请确认：（1）您是否在当天（以 HKT
为准）已登录；（2）抽奖机会是否已经显示在活动页面中（有时有短暂延迟）；（3）今日是否已经领取过。如果确认满足条件但未到账，请联系客服核查。

**Q：每日抽奖次数有上限吗？我今天抽了很多次，有些机会没法用了？**
A：部分抽奖活动确实设有每日抽奖次数上限（如每日最多抽 3 次）。超过当日上限后，即使您还有剩余抽奖机会，当天也无法继续使用，需等到次日（以
HKT 时间为准）再使用。剩余机会不会消失，可在次日继续使用。

**Q：我中了实物奖品，要在哪里填收货地址？多久能送到？**
A：中实物奖品后，请在活动页面或「活动中心 →
我的奖励」找到对应奖励，按提示填写收货信息（姓名、手机号、收货地址）。收货信息填写通常有截止时间，逾期未填写可能导致奖品被取消。填写完成后平台会安排寄出，具体物流时效以活动说明为准，一般需要
7-15 个工作日，跨境寄送可能更长。如超时未收到，请联系客服查询。

**Q：我和好友一起参加分享抽奖，好友没有完成助力，我的抽奖次数还能增加吗？**
A：分享抽奖中，好友需满足活动设定的助力条件（如点击链接、完成注册等）才算有效助力，助力成功后您和好友的抽奖次数/奖励才会同步增加。好友仅点击但未完成所需操作，不视为有效助力，不会增加您的抽奖次数。请提醒好友按活动要求完整完成助力步骤。

## 抽奖资格与合规要求

- 每次抽奖均消耗机会，无论结果如何，机会不可找回
- 活动结束后未使用的抽奖机会自动失效，不做补偿
- 中奖概率在活动详情页有公示，如对抽奖公平性有疑问可联系客服
- 账户需完成 KYC 才能领取中奖奖励
- 所有活动时间以香港时间（HKT，UTC+8）为准，每日抽奖机会通常以 00:00 HKT 为界重置
- 中实物奖品须在截止时间内填写收货地址，逾期可能导致奖品被取消
- 奖励查看路径：APP「活动中心 → 我的奖励」
