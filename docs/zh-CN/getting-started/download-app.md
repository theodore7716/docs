# 下载 App

长桥 App 支持安卓、iOS、Mac 及 Windows 四个平台，本页提供各平台下载链接与安装说明。

> 以下下载链接的版本不一定是最新版本，安装后可在 App 内自行检查更新。

## 安卓

- [HK App 下载链接](https://static.lbkrs.com/app/longbridge-online-release.apk)
- [SG App 下载链接](https://static.lbkrs.com/app/longbridgesg-online-release.apk)

## 苹果（iOS）

长桥 iOS App 已在中国大陆 App Store 下架，中国大陆地区用户无法通过中国大陆 Apple ID 下载。海外其他地区用户可使用当地 Apple ID 在 App Store 正常搜索并下载。

### 中国大陆通过 TestFlight 下载

- HK App：使用大陆 Apple ID 下载 TestFlight，[点击加入](https://testflight.apple.com/join/q27PHAxw)后选择长桥 App 安装。

  ![TestFlight 安装长桥 HK App](./images/testflight-hk.png)

- SG App：使用其他地区 Apple ID 下载 TestFlight，[点击加入](https://testflight.apple.com/join/WVQMKEyl)后选择长桥 App 安装。

  ![TestFlight 安装长桥 SG App](./images/testflight-sg.png)

### 更新已安装的 App

打开【设置】→【App Store】→ 开启【App 更新】，即可自动更新。也可在 App Store 右上角头像 →【账户】页面下拉刷新后，进入【App】或【已购项目】手动更新。

### 在「已购项目」中找不到长桥

1. **App 被隐藏**：App Store → 右上角头像 → 账户 → 顶部账户名 →【账户设置】→【隐藏的已购项目】，找到"长桥"后点击【解除隐藏】；
2. **Apple ID 不匹配**：当前登录的 Apple ID 不是当初下载时使用的账号，切换回原 Apple ID 后重试。

## 中国大陆存量用户说明

长桥已停止接受中国大陆境内用户通过存量证明方式开户。中国大陆境内已有账户的存量用户可继续正常使用长桥客户端进行投资交易，相关服务不受影响。

访问长桥官网时可能遇到卡顿或受限，建议直接通过 App 使用。如需客服支持，可通过 App 内【我的】-【帮助与客服】联系，或致电 +852 3851 1777。
