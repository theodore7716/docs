# 新股认购常见问题

## 普通认购

### 普通认购使用哪部分资金？

普通认购可使用账户持仓抵押融资及可用现金进行认购申请。可用现金需要扣减出金提交、买入股票金额、买入基金等所有业务的交易款。

认购扣款日进行扣款时，若账户相应可提取金额不足，将会产生借币融资，按融资利率计息。

### 普通认购什么时候扣款？

认购资金将在发起申请后实时冻结，并于认购截止日当天进行扣除。

### 普通认购公布中签结果后，资金如何处理？

- **未中签**：扣除的认购额本金在公布中签日退回账户
- **全部中签**：增加持仓，暗盘时间可正常交易
- **部分中签**：中签部分增加持仓，未中签的多扣部分资金在公布中签结果后退回账户

### 普通认购可以修改和撤销吗？

在认购截止前，支持改单和撤销。入口：长桥 App → 市场 → 新股 → 认购记录。

> 为避免撤销后未能成功重新申请，建议留意认购页面的截止时间。

## 银行融资认购

### 账户中必须有港币现金吗？

银行融资认购最大融资比例为认购额的 97%，因此账户中至少需要有 **3% 的港币现金**。

当账户港币现金不足时，系统会使用账户持仓股票作为抵押进行借款。若账户港币现金不足且持有股票不足以抵押，可进行港币入金或货币兑换后提交认购订单（融资系数将上浮，不同于普通股票交易融资系数）。

### 本金部分什么时候扣款？

银行认购额中本金部分将在发起申请后实时冻结，并于银行融资认购截止日进行扣除。

### 融资部分什么时候扣款？

提交银行融资认购时，银行融资的金额先进行记账，等到公布中签结果当日再进行扣款。

### 银行融资利息如何计算？和申请早晚有关系吗？

计息公式：日利息 = 融资额 ×（银行融资利率 ÷ 365）

- **开始计息日**：新股认购截止日（与申请早晚无关）
- **截止计息日**：公布中签结果前一天，于公布中签当日扣除

如遇新股延期或取消上市，计息时长相应增加。

### 申请了银行融资认购后存入资金，怎么处理还款？

分两种情况：

- **券商借币融资部分**（扣款日账户可提取金额不足本金所需产生的借款）：资金入账后自动还款，但须与借款币种相同，不同币种需换汇后还款。
- **银行融资部分**（本金以外最高 97% 的借款）：**不能提前还款**，将在公布中签日依据中签情况进行实际扣款。扣款时若账户余额足够，则不会继续产生融资。

### 银行融资认购公布中签后出现 Margin Call，怎么处理？

若账户保证金比例不足，会触发 Margin Call，系统将向您发出追缴保证金通知。务必在**上市交易日下午 14:00 前**存入足额保证金，否则有权在未通知的情况下对中签新股进行平仓。

### 银行融资利息什么时候扣除？

银行融资利息在公布中签结果当日扣除。

### 融资比例必须使用最大倍数吗？

不是。系统可能同时开放不同融资比例的银行融资认购，您可以根据开放情况及自身需求自行选择。

### 银行融资认购可以修改和撤销吗？

在认购截止前，支持改单和撤销。入口：长桥 App → 市场 → 新股 → 认购记录。

> 为避免撤销后未能成功重新申请，建议留意认购页面的截止时间。

## 中签结果查询

认购中签结果将于上市前一个交易日下午 16:00 前公布，请留意 App 消息通知，也可在消息中心 → 交易变动和认购记录详情中查询。

香港身份证明文件开户的客户可通过披露易网站查询；其他国家/地区的客户可能因身份证明文件问题，无法通过披露易查询分配结果。

## 其他常见问题

### 普通认购和银行融资认购可以同时申请吗？

不可以，只能二选一。

### 当天入金多久可以用来打新？

当天存入的资金，需等到银行到账并资金加入账户后方可使用（部分银行可能次日到账）。若账户存在欠款，存入资金会优先偿还欠款，欠款还清后可使用持仓抵押打新（是否可用取决于账户风控情况）。

### 当天卖出股票，资金多久可以用来打新？

当天卖出的股票可用于认购下单。卖出资金通常在 T+2 交易日结算，若扣款日结算已完成则直接用于缴纳认购款项；若未清算完成，则会产生借币融资。

### 认购越多，中签概率越高吗？

不是。港股打新由抽签决定，规则由港交所制定，认购数量不影响中签概率。

### 不中签的话，认购手续费可以退回吗？

认购手续费无论是否中签均需缴纳，在认购截止当日扣除。如遇新股延期、取消上市等情况，认购手续费同样不退还。

### 为什么打新认购会产生汇损？

当账户可用的港币现金不足以支付认购款项时，系统可能自动将其他币种换汇为港币进行还款。由于换汇依市场汇率执行，汇率波动可能导致实际换汇金额与预期有所差异，从而产生汇损。

### 中签费用如何构成？

中签后收取总计 **1.0085%** 的费用，明细如下：

| 费用项目 | 费率 |
|---------|------|
| 佣金（Commission rate） | 1% |
| 交易会费（Transaction levy） | 0.0027% |
| 交易费（Trading fee） | 0.00565% |
| 财汇局交易征费（FRC Transaction levy） | 0.00015% |
| **合计** | **1.0085%** |
