# 交易挑战

交易挑战活动（Trade Challenge Activity）是 LongBridge
平台推出的任务型交易活动。用户在活动期间接受系统设定的交易挑战——如在指定时间内完成特定的交易次数或达到特定资产水平——完成挑战后获得奖励。

## 挑战类型

交易挑战分为两大类型：

**达标型挑战**：完成绝对数量/金额目标

- 交易次数达标（TradeTimes）：本月/本季度成交达到指定笔数
- 日均资产达标（DailyAssets）：本月/本季度日均资产达到指定金额

**提升型挑战**：相比上个月/季度有所提升

- 交易次数提升（TradeTimesIncrease）：本期成交笔数超过上期
- 日均资产提升（DailyAssetsIncrease）：本期日均资产超过上期

> 提升型挑战中，活动详情页会显示您的「上期完成值」作为参考基准。

## 挑战周期

- 月度挑战（Month）：以自然月为统计周期
- 季度挑战（Quarter）：以自然季度为统计周期

## 参与条件

- 持有有效的 LongBridge 证券账户，账户状态正常
- 已完成实名认证（KYC）
- 账户所属地区/持牌人在活动覆盖范围内
- 部分挑战活动可能要求账户有一定的交易历史或资产门槛

## 任务类型

- 单笔挑战：完成一笔特定条件的交易（如单笔买入港股 10,000 港币）
- 多笔挑战：在活动期间完成指定笔数的有效成交
- 连续交易挑战：连续多天均有有效成交记录
- 品种挑战：交易指定市场（港股/美股/ETF）的指定品种
- 金额挑战：累计成交金额达到指定门槛
- 阶梯挑战：完成不同难度级别的交易目标，解锁对应档位的奖励

## 参与步骤

1. 在「活动中心」找到当期交易挑战活动
2. 阅读挑战规则，了解需要完成的具体交易要求
3. 在报名时间内完成报名（部分活动有专门报名阶段，报名过期则无法参与）
4. 在活动有效期内按要求完成交易操作
5. 系统每天香港时间 12:00 批量更新挑战进度
6. 挑战结束后系统进行结算（Settling 阶段，约 1 天），确认最终结果
7. 结算后在 APP 内主动领取奖励（需在领奖时间窗口内操作）
8. 在「活动中心 → 我的奖励」查看奖励状态

## 任务完成判定

- 以最终有效成交为准，委托未成交（挂单状态）不计入挑战进度
- 已撤销的委托不计入挑战进度，撤单会影响挑战完成情况
- 废单（系统拒绝的委托）不计入
- 活动期间内完成的成交才算有效
- 挑战进度每天香港时间 12:00 批量计算（计算前一日数据），当天交易记录将在次日 12:00 后反映在进度中，非实时更新

## 挑战进度档位

- 未达标：尚未完成挑战目标，或因撤单等原因进度回退至零
- 部分达标：曾完成部分进度，但因撤单导致部分成交被剔除，进度存在波动（尚未完全失效）
- 全部达标：所有挑战目标均已达成，奖励可领取

## 挑战阶段说明

| 阶段    | 含义                 | 用户操作                 |
|-------|--------------------|----------------------|
| 报名未开始 | 活动尚未进入报名时间         | 等待报名开放               |
| 报名中   | 可在此阶段完成报名          | 尽快报名，截止后无法参与         |
| 报名成功  | 已成功报名，等待挑战期开始      | 等待挑战期开始              |
| 报名过期  | 报名时间结束前未报名         | 无法参与本期，等待下一期         |
| 挑战中   | 正在进行交易挑战           | 完成交易任务，进度每天 12:00 更新 |
| 结算中   | 挑战期结束后约 1 天，系统最终结算 | 等待结算完成               |
| 挑战成功  | 挑战目标达成             | 前往活动页面领取奖励           |
| 挑战失败  | 挑战目标未达成            | 无需操作，等待下一期           |
| 领取奖励  | 奖励可领取阶段            | 在时间窗口内点击「领取」         |
| 已领奖   | 奖励已成功领取            | 在「我的奖励」查看状态          |
| 奖励失效  | 领奖时间窗口已过           | 奖励自动失效，不可补领          |

## 奖励类型

- 现金奖励：发放至账户可用现金余额
- 交易券：用于抵扣手续费
- 体验金：有使用期限和条件的模拟资金
- 勋章/成就：平台荣誉展示，部分可兑换实际奖励

## 发放时机

交易挑战奖励需要在 APP 内主动领取，不会自动发放到账户。活动结束并经过结算期（约 1
天）后，奖励进入可领取状态。活动页面会出现「领取奖励」按钮，点击后触发发放。奖励领取有时间限制：若超过领奖截止时间未操作，奖励将进入「奖励失效」状态，无法补领。具体领奖时间窗口以活动详情页为准。

## 奖励状态说明

| 状态    | 含义             | 用户操作        |
|-------|----------------|-------------|
| 待发放   | 挑战已完成，奖励排队等待处理 | 无需操作，等待系统处理 |
| 发放中   | 系统正在将奖励发送至账户   | 等待，通常数小时内完成 |
| 已发放   | 奖励已成功到账        | 可在账户中查看使用   |
| 锁定中   | 奖励已记录但处于锁定期    | 等待锁定期结束自动解锁 |
| 已解锁可用 | 锁定期结束，可正常使用    | 正常使用奖励      |
| 发放失败  | 奖励发放失败         | 联系客服处理      |
| 已取消   | 奖励因规则原因取消      | 联系客服确认原因    |

## 撤单对挑战的影响

若用户撤销了已计入挑战进度的交易委托，该笔成交记录将被剔除。挑战进度会相应回退，可能导致原本已达成的里程碑失效。若因撤单导致挑战进度不足，已发放的对应奖励可能被追回（以活动规则为准）。建议在确认交易意愿后再下单，避免撤单影响挑战进度。

## 连续挑战中断

连续交易挑战（如「连续 7 天交易」）中如某天无有效成交，连续计数将中断。中断后通常需要重新从第 1
天开始累计，以活动规则为准。建议每天在交易时段内完成至少一笔有效成交以保持连续进度。

## 挑战达标与奖励发放答疑

**Q：我的挑战进度显示完成了，但奖励还没到账，是怎么回事？**
A：交易挑战奖励不会自动到账，需要您主动领取。请按以下步骤操作：（1）等待活动结束后的结算期（约 1
天）；（2）回到活动详情页，查看是否出现「领取奖励」按钮；（3）点击领取按钮触发发放。请注意领奖有时间限制，超过截止时间后奖励将失效，无法补领。如果已经点击领取但奖励仍未到账（通常
1-3 工作日），请联系客服。

**Q：挑战成功了但我忘记领取奖励，现在奖励失效了怎么办？**
A：奖励一旦进入「奖励失效（RewardExpired）」状态，通常无法恢复。建议活动结束后及时关注活动页面或站内通知，在领奖时间窗口内完成领取。如有特殊情况（如系统故障导致无法领取），请联系客服提供相关截图，客服会协助核查是否可补发。

**Q：我下了委托单但撤销了，这会影响我的挑战进度吗？**
A：是的，撤单会影响挑战进度。交易挑战以最终有效成交为统计依据，撤销的委托不计入挑战完成度。如果您撤销了已计入进度的成交，挑战进度会相应扣减。建议确认交易意愿后再提交委托，尽量避免不必要的撤单。

**Q：挑战任务里要求买入港股，我买了 ETF 算吗？**
A：取决于活动的具体规则。部分活动明确限定了品种类型（如「港股正股」），ETF
可能不在范围内；部分活动则对港股市场所有品种都有效。请仔细阅读活动详情页中的「有效交易品种」说明，如有疑问可咨询客服确认。

**Q：挑战活动中途取消参与，之前的交易进度会保留吗？**
A：通常情况下，挑战进度与账户绑定，取消参与后如重新参与可能会重置进度，也可能保留之前的进度，具体以活动规则为准。如果您只是暂时不想继续，可以不做任何操作，不参与交易，进度会保留到活动截止时间。

**Q：奖励显示「发放失败（Failed）」怎么处理？**
A：发放失败的常见原因：（1）账户 KYC
未完成或审核失败；（2）账户存在限制或异常；（3）系统技术问题。请先确认账户实名认证已完成且账户状态正常，然后联系客服，提供您的账户信息和奖励截图，客服会协助排查并重新触发发放。

**Q：我同时参与了多个挑战活动，交易进度会互相干扰吗？**
A：同期多个挑战活动通常各自独立统计进度，同一笔成交可能同时计入多个活动的进度（取决于每个活动的统计规则）。如需了解某笔交易是否同时计入多个活动，可在各活动详情页查看规则说明，或联系客服确认。

**Q：挑战活动的进度实时更新吗？成交后要多久才看到？**
A：交易挑战进度不是实时更新的。系统每天香港时间 12:00 批量计算前一日的交易数据，因此当天完成的交易需要等到次日 12:00
后才会反映在挑战进度中。如果今天完成了成交，当天进度可能暂时未变，属正常现象。

**Q：提升型挑战中，「上期完成值」是怎么计算的？**
A：对于提升型挑战（如交易次数提升、日均资产提升），系统会以上个月（月度挑战）或上个季度（季度挑战）的实际完成值作为基准。您需要在本期超过这个基准值才能完成挑战。活动详情页会显示您的上期完成值作为参考。

**Q：我之前参加过交易挑战，这次又出了一个，我还能参加吗？**
A：一般情况下可以参加，不同期次的挑战活动各自独立，参与过上一期不影响参与新一期。但需注意：（1）部分挑战限制同一类型同时只能参与一个，须先完成或退出当前挑战才能加入新的；（2）部分挑战针对「首次操作」（如首次入金、首次完成某类交易），已完成过该操作的用户不再满足触发条件。具体以新活动的详情页说明为准。

## 参与资格与挑战规则限制

- 挑战奖励需主动领取，逾期失效，请勿拖延
- 有报名阶段的挑战须在报名时间内完成报名，否则无法参与
- 撤单会影响挑战进度，确认交易意愿后再下单
- 挂单（委托未成交）不计入挑战进度，必须是实际成交
- 使用体验金进行的交易通常不计入挑战进度
- 所有活动时间以香港时间（HKT，UTC+8）为准
- 奖励查看路径：APP「活动中心」活动详情页领取，或「我的奖励」查看状态
