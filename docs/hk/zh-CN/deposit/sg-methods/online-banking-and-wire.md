# 网银或电汇入金与出金

通过手机银行或网上银行将资金转账至长桥新加坡收款账户，或通过电汇方式从长桥新加坡账户提取资金。新加坡本地转账与跨境电汇均通过此方式操作，入金时需上传汇款凭证。

## 入金

| 项目 | 说明 |
|------|------|
| 支持币种 | 新加坡元（SGD）、美元（USD）、港元（HKD） |
| 预计到账时间 | 预计 1–3 个交易日 |
| 手续费 | 长桥及代理银行（DBS）免费；汇款银行和中转银行可能收取约 200 HKD 或 25 USD |

## 收款银行信息

|------|------|

| 收款人账号 | 0725885663 |

| 附言（汇款备注） | 填写证券账户号（如 SG10039100），**必填** |

费用承担方：SHA（各环节各自扣除）

## 操作步骤

1. 打开长桥 App，进入**资产 → 存入资金 → 选择入金币种**，查看收款银行信息
2. 在银行手机端或网页端完成转账操作
3. 保留汇款凭证截图
4. 返回长桥 App，上传汇款凭证

> 完成转账后请立即上传凭证，否则影响入金进度。

## 特殊情况说明

**iFAST Global Bank（奕丰环球银行）**：属于跨境电汇，因使用第三方支付平台，收款行可能无法解析付款方信息，需在长桥上传 iFAST 开户证明及汇款流水。

**IBAN**：International Bank Account Number，长度通常为 15–34 个字母数字字符，全球已有 70 多个国家使用，主要集中在欧洲、中东和加勒比地区。海外银行卡可在银行账户处直接输入 IBAN Code，不了解 IBAN 可查询银行结单获取。

**三位数银行代码**：主要用于香港地区，海外银行卡没有银行代码时可填写 000。

**中转行显示问题**：跨境电汇资金可能显示为中转行账户，如遇此情况请联系客服。

## 同名账户要求与中转行说明

- 转账银行账户名必须与长桥证券账户名同名，不可使用他人或联名账户
- 银行通知「已汇出」不等于长桥已收款，资金到达后需结算与审批
- 节假日不处理汇款业务，请预留时间
- 不接受直接存入现金

## 出金

### 出金操作步骤

登录长桥 App，进入**资产 → 全部功能 → 提取资金**：

1. 选择货币
2. 选择收款银行
3. 输入金额
4. 提交请求

### 新加坡银行出金费用

| 货币 | 提款金额 | 收款银行/方式 | 预计抵达时间 | 长桥手续费 | 预估银行收费 |
|------|----------|---------------|--------------|-----------|--------------|
| SGD | 低于 200,000 | 新加坡银行（FAST 转账） | 1 个工作日 | 免费 | 无 |
| SGD | 低于 200,000 | 其他 | 1–3 个工作日 | 免费 | 无 |
| SGD | 高于 200,000 | DBS / POSB 银行 | 1–3 个工作日 | 免费 | 无 |
| SGD | 高于 200,000 | 其他 | 1–3 个工作日 | SGD 10（豁免） | 约 SGD 10 |
| USD / HKD | 任何金额 | DBS / POSB 银行 | 1–3 个工作日 | 免费 | 无 |
| USD / HKD | 任何金额 | 其他 | 1–3 个工作日 | SGD 10（豁免） | 约 USD 35–60 等值 |

USD 出金长桥不收费，可能由银行方收费。

### 非新加坡银行出金费用

| 货币 | 提款金额 | 预计抵达时间 | 长桥手续费 | 预估银行收费 |
|------|----------|--------------|-----------|--------------|
| SGD / USD / HKD | 任何金额 | 1–3 个工作日 | SGD 10（豁免） | 约 USD 35–60 等值 |

以上仅为长桥证券的收费，中转行、收款行可能有额外费用。

### 出金前置条件与同名要求

- 提款资金须转入与长桥账户同名的银行账户，不可提取至他人账户、第三方支付平台或联名银行账户
- 长桥新加坡账户支持 SGD、USD、HKD 出金，请确认收款银行账户支持对应币种
- 银行账户须先添加并完成认证后才能用于出金
- 仅可提取已结算金额，且账户现金净余额需大于 0
- 汇款费用如有变更，恕不另行通知；汇款行、收款行及中间行产生的费用由客户承担
