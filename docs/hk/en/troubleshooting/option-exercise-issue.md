# 期权行权异常怎么办

期权到期或行权后出现预期外情况时，按以下场景排查。

> **前提说明**：长桥对期权行权的处理规则——价内期权（ITM）到期自动行权，价外期权（OTM）到期自动作废。**不支持主动提前行权，也不支持主动放弃行权**。如需避免行权，须在到期日收盘前平仓卖出期权合约。

## 持有的期权到期后未被行权

**可能原因**：期权在到期日收盘时处于价外（OTM）或平价，系统判定放弃行权。

**核查方式**：

1. 前往「资产 → 全部功能 → 订单记录」，找到该期权合约的到期处理记录
2. 确认到期日收盘时期权的行权价与正股价格关系：
   - 看涨期权（Call）：正股价 > 行权价 → 价内，应行权
   - 看跌期权（Put）：行权价 > 正股价 → 价内，应行权
   - 差值低于 0.01 USD 时，是否行权由结算所决定，长桥不干预

**如果确认期权应为价内但未被行权**：联系客服并提供合约代码和到期日期，由客服核查结算记录。

## 账户出现了预期外的正股持仓

期权行权后，会以行权价买入或卖出对应数量的正股（每张合约 = 100 股）。若账户出现正股持仓但未预期：

- **买入看涨（Long Call）行权**：账户新增买入正股持仓，成本为行权价
- **买入看跌（Long Put）行权**：账户正股持仓减少（卖出正股），如原本无正股则可能形成空头仓位
- **行权结算时间**：期权 T+1 交收，行权产生的正股仓位在下一个交易日开盘前到账

如不需要行权产生的正股，须在开盘后手动卖出。

## 持有的卖出期权（Short）被提前行权

卖出期权（Short Options）即使尚未到期，对手方有权提前行使权利（美式期权），你将被强制以行权价买入或卖出正股。以下情况触发概率较高：

- 期权深度价内
- 正股即将除息（持有看涨期权的对手方可能选择提前行权以获取股息）
- 正股处于停牌状态

被提前行权后：

1. 检查「订单记录」确认被行权的合约和数量
2. 若产生了不需要的正股持仓，可在市场开盘后卖出
3. 若账户保证金因被行权而触发风控，需及时补充资金

## 末日期权被强制平仓

当日到期的期权（末日期权），在到期日美东时间 11:00–14:00 期间若期权变为价内，长桥会加收额外保证金。如账户保证金不足，系统可能在收盘前强制平仓该期权。

**避免方式**：在到期日开盘后尽早处理末日期权，不要等到下午才操作。

## 行权后账户数据未更新

行权处理完成后，持仓和资产显示可能有延迟（通常在下一个交易日开盘前更新完毕）：

1. 下拉刷新「资产」页面
2. 切换至「订单记录」查看行权处理状态
3. 如次日开盘后数据仍未更新，联系客服核查

## 联系客服

准备以下信息：

- 期权合约代码（如 `AMD260417C195000`）
- 到期日期
- 合约张数
- 异常描述（未行权 / 多出持仓 / 被提前行权等）

**在线客服**：长桥 App → 我的 → 帮助与客服 → 在线客服
