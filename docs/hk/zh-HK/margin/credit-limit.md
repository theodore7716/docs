# 融资额度

融资额度是账户可借入资金的上限，根据存入的现金或转仓资产自动评估，无需手动申请初始额度。结欠金额不得超过融资额度上限。

长桥账户默认为融资账户（保证金账户），支持杠杆交易。如不需要杠杆，可降低或关闭融资额度，以避免误操作产生利息。

## 提高融资额度

融资额度根据账户存入的现金或转仓资产自动评估。若需申请更高融资限额，可通过存入资金或将股票转仓至长桥账户来增加账户资产净值，系统将据此重新评估并更新额度。

## 降低融资额度

1. 打开长桥 App，进入「资产」页面
2. 找到「融资状态」或「我的融资额度」
3. 点击「调整额度」，将额度调低至所需金额
4. 确认提交，24 小时内生效

每 24 小时内最多申请 2 次（新加坡账户每 24 小时限 1 次）。

降低额度不影响已有持仓，仅限制新增融资买入。如需重新开启融资，按相同流程申请提高额度即可。

## 关闭融资额度

将融资额度调整为 0 即等同关闭融资。关闭后账户仍为保证金账户，但不可使用杠杆买入。

**前提条件**：账户内无未平仓的融资持仓（即无结欠），且无待成交的融资买入委托。如有融资持仓，需先卖出并还清欠款后再申请关闭。

**确认是否有结欠**：App → 资产 → 现金页，查看「融资余额」或「结欠」字段：

- 显示 0 或为空：无结欠，可直接关闭
- 显示负数金额：需先还清。入金后系统将于下一结算日自动扣除，或通过换汇偿还不同币种的结欠

## 降低额度对持仓的影响

- 降低额度不影响已有持仓，仅限制新增融资买入
- 持仓市值下跌导致维持保证金不足时，平台仍会发出追缴通知，与是否主动降低额度无关
