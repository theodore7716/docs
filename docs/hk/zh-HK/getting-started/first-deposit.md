# 完成首次入金

在本教程结束时，你的长桥账户余额将显示港元资金，可以用于购买港股。

完成这个教程大约需要 **10 分钟操作**，加上等待银行转账处理的时间（港股交易日内约 2 小时）。

## 开始前的准备

确认以下三项已就绪：

- 长桥账户已开通（可正常登录）
- 已在长桥 App 绑定一张香港银行卡
- 银行账户中有足够余额

如果银行卡尚未绑定，请先在 App 中添加：**资产 → 银行卡 → 添加银行卡**。

## 第一步：进入存入资金页面

打开长桥 App，点击底部导航栏的**「资产」**。

你会看到账户总览页面，顶部显示资产总额（目前为 0 或你已有的余额）。

点击**「存入资金」**按钮。

## 第二步：选择入金方式

进入存入资金页面后：

1. 在币种选择中点击**「港元 HKD」**
2. 在下方选择你的香港银行卡
3. 点击**「FPS 转数快」**

你会看到一个 FPS 识别码（一串数字，如 `169152691`）。点击旁边的**「复制」**按钮。

> 这个 FPS ID 就是长桥的收款地址，你接下来要把它转到你的银行 App 里使用。

## 第三步：在银行 App 发起转账

现在切换到你的银行 App（不要关闭长桥 App，可以返回桌面）。

在银行 App 中：

1. 进入**转账**或**FPS 转数快**功能
2. 选择**「以 FPS 识别码转账」**
3. 粘贴刚才复制的 FPS ID
4. 输入转账金额（例如 5,000 港元）
5. 确认收款人名称显示为 **Long Bridge HK Limited**，然后提交转账

注意：收款人名称必须是 Long Bridge HK Limited 才代表转账地址正确。

银行 App 显示「转账成功」后，**立即返回长桥 App**。

## 第四步：上传转账凭证

返回长桥 App 的 FPS 转数快入金页面（如果不在该页面，重新进入**资产 → 存入资金 → 港元 → FPS 转数快**）。

你会看到一个上传凭证的区域：

1. 点击上传图标
2. 选择刚才银行 App 的转账成功截图
3. 确认金额和 FPS ID 可见
4. 点击**「提交」**

> 凭证必须在转账后立即上传，拖延会影响入金审核进度。

提交后，你会看到入金申请状态变为**「处理中」**。

## 等待资金到账

现在你只需要等待。

**如果是港股交易日（周一至周五 09:00–23:59）**：资金通常在 2 小时内到账。

**如果在其他时段或香港公众假日**：资金会在下一个港股交易日上午 09:40 前到账。

你可以在**资产 → 存入资金 → 入金记录**里随时查看处理进度，状态会从「处理中」变为「已入账」。

## 确认完成

当入金记录状态显示「已入账」，回到**「资产」**页面。

你会看到港元余额已更新，显示你刚才存入的金额。

**恭喜！你已完成第一次入金。** 这笔资金现在可以用于购买港股。
