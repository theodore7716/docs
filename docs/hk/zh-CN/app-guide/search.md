# 搜索

搜索是 App 内的统一检索入口，点击任意页面右上角的搜索图标（🔍）进入。支持股票标的、内容资讯和功能工具的综合搜索。

## 搜索框

输入框支持：

- **股票代码**：如 `00700`、`AAPL`、`AMD260417P195000`（期权合约）；港股代码支持补零识别，输入 `700` 或 `00700` 均可找到腾讯控股
- **公司名称**：中英文均可，如「腾讯」「Apple」
- **投资相关问题**：自然语言提问，由 LongbridgeAI 解答

期权合约可直接输入完整代码（如 `AMD260417P195000`），也可先输入正股代码（如 `AMD`），进入详情页选择「期权链」标签查看所有合约。搜索历史记录右侧有清除图标，可单条删除或一键清空全部历史。

## LongbridgeAI

搜索页顶部固定展示 LongbridgeAI 对话入口，可直接向 AI 提问投资相关问题，例如：

- 「最近有哪些值得关注的港股？」
- 「解释一下市盈率（P/E）」
- 「这只股票的基本面怎么样？」

> LongbridgeAI 与普通搜索的区别：普通搜索返回匹配的股票、新闻、工具等结构化结果；LongbridgeAI 回答开放性问题、解释概念、提供分析，适合需要解读的场景。

## 快捷工具入口

搜索框下方横向滚动的功能快捷入口：

| 工具    | 说明                   |
|-------|----------------------|
| 期权榜   | 当日期权成交活跃排行           |
| 美股定投  | 设置美股定期定额买入计划         |
| 桌面端   | Longbridge Pro 客户端介绍 |
| 美股排行榜 | 涨跌幅、成交量等多维排行         |
| 股票 VS | 两只股票横向对比             |
| 热门股单  | 当前热门主题股单             |
| 深度数据  | 机构研报入口               |
| 选股器   | 多条件股票筛选              |
| 财经日历  | 经济数据与财报发布日程（点击事件旁的铃铛图标可设置提醒，触发时 App 推送通知） |
| 货币基金  | 余额通/货币基金入口           |
| 结构化产品 | 雪球、票据等产品             |
| 私募基金  | 私募产品入口               |

## 热门股票

页面中部展示港股和美股当日热门标的排行，显示排名、代码、名称和最新涨跌幅，点击直接跳转标的详情页。

## 搜索结果

输入关键词后，结果页分类展示：

- **股票卡片**：名称、代码、最新价、涨跌幅，点击进入详情页
- **相关资讯**：与该股票相关的新闻文章
- **相关股单 / 用户**：持有或讨论该标的的股单和社区用户
