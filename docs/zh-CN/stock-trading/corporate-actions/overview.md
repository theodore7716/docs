# 公司行动

本章节说明长桥对各类公司行动的处理规则，包括现金分红、特殊派息、股票股利（送股）、供股（配股）、选股选息、拆股与合股的操作流程与注意事项。

## 为什么要了解公司行动

当你持有的股票发生**分红**、**配送股**、**拆合股**等事件时，这些都是公司行动。长桥会自动处理大部分动作（如现金分红自动入账），但某些行动（如供股、选股选息）需要你**主动应对**——否则可能失去收益或承受其他影响。

**关键场景**：
- 你持有的股票有分红 → 需要了解如何收取，是否有税务影响
- 公司送股或配股 → 需要了解你是否有权利、如何行使
- 公司拆股或合股 → 持仓数量会变，需要知道交易有什么影响
