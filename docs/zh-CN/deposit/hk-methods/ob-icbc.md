# 工银亚洲网银转账

通过工银亚洲手机银行或网上银行将资金转至长桥，转账完成后上传凭证即可。

> 网银转账的到账时间、手续费及通用注意事项，见 [网银转账入金](/deposit/hk-methods/online-banking-transfer)。

## 操作步骤

1. 打开**长桥 App** → **资产** → **存入资金**，选择入金币种，选择香港银行卡，选择**网银转账**，查看收款银行信息

   ![长桥 App 查看收款信息](https://pub.pbkrs.com/uploads/2026/0a2f1f2d82a1920620134bdb9f5c6658)

2. 通过工银亚洲手机银行或网上银行，完成转账并截图保留凭证

3. 立即返回**长桥 App** → **资产** → **存入资金** → **网银转账**，上传汇款凭证

   ![上传汇款凭证](https://pub.pbkrs.com/uploads/2026/8a008421c0226d94abf1f2b5234c1bba)

   > 凭证必须在汇款完成后立即上传，否则影响入金进度。
