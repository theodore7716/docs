# 市场行情

市场行情页展示各地区股票市场的指数走势、涨跌分布和热门股票排行，支持港股、美股、沪深、沪深港通和新加坡五个板块。

## 页面入口

底部导航「市场」→ 顶部切换「市场」标签 → 左右滑动切换板块。

## 各板块说明

### 港股

展示港股市场的主要指数（如恒生指数、国企指数），以及市场整体涨跌分布和热门排行。

### 美股

展示美股三大指数（道琼斯、纳斯达克、标普 500），并提供市场温度仪表盘。

### 沪深

展示 A 股三大指数（上证指数、深证成指、创业板指），并提供市场温度仪表盘。

### 沪深港通

专门展示南北向资金流向数据，包含港股通（沪）和港股通（深）两个方向的资金明细。

### 新加坡

展示新加坡市场主要指数（富时海峡指数、富时海峡 REITs、富时海峡中国）及市场数据。

## 通用功能模块

各板块页面包含以下通用模块：

### 指数概览卡片

以卡片形式展示该板块主要指数，每张卡片包含：

| 字段    | 说明        |
|-------|-----------|
| 指数名称  | 该市场的代表性指数 |
| 最新点位  | 实时更新的指数点位 |
| 涨跌幅   | 当日涨跌百分比   |
| 日内走势图 | 当日分时走势缩略图 |

点击卡片进入该指数详情页。

### 市场温度

美股、沪深、新加坡板块提供市场温度仪表盘，从估值和情绪两个维度综合评估当前市场状态：

| 字段   | 说明                            |
|------|-------------------------------|
| 温度值  | 0°~100° 之间，数值越高代表市场越热         |
| 估值判断 | 当前市场整体估值水平（偏低 / 正常 / 偏高 / 极高） |
| 情绪判断 | 当前市场情绪状态（冰冻 / 平稳 / 高涨等）       |
| 状态描述 | 综合描述当前温度趋势                    |

点击市场温度卡片可查看详细分析报告。

### 涨跌分布

以横向柱状图展示当日全市场股票的涨跌家数分布，分为：

- **上涨**：当日上涨的股票数量
- **平盘**：当日持平的股票数量
- **下跌**：当日下跌的股票数量

### 市场异动

实时推送当日出现异常波动的股票：

| 字段        | 说明           |
|-----------|--------------|
| 股票名称 / 代码 | 异动股票的名称和交易代码 |
| 异动类型      | 如急速拉升、加速下跌等  |
| 涨跌幅       | 当前涨跌百分比      |
| 时间        | 异动发生时间       |

支持切换「市场异动」和「自选」标签，后者只显示用户自选股中的异动。

### 热门股票排行

展示当前市场热度最高的股票排名，支持按时间维度筛选：

- 日内 / 5 分钟 / 半年 / 今年以来 / 创 52 周新高

## 沪深港通资金流向

沪深港通板块专项展示南北向资金数据：

### 南向资金（港股通）

内地资金流入香港市场的数据：

| 字段    | 说明                     |
|-------|------------------------|
| 成交净买入 | 当日净买入金额（正值为净买入，负值为净卖出） |
| 实时净流向 | 实时更新的资金净流向             |
| 剩余额度  | 当日可用额度剩余情况             |

分港股通（沪）和港股通（深）分别展示明细。

### 北向资金（沪深股通）

境外资金流入 A 股市场的数据，字段结构与南向资金相同。

### 历史走势图

以柱状图展示近期每日成交净买入数据，直观呈现资金流向趋势。

## 可执行操作

- 点击指数卡片 → 进入指数详情页
- 点击市场温度 → 查看估值与情绪详细分析
- 切换异动 / 排行时间标签 → 过滤显示范围
- 点击个股 → 进入该股票行情详情页
- 点击右上角「关注」→ 快速添加自选股
