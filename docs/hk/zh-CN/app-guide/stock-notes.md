# 股单

股单是由平台官方或用户创建的主题股票组合（类似投资主题篮子）。关注股单后可一键查看组合内所有标的的实时表现。入口在 App 底部导航「关注」Tab 的「股单」子页。

## 股单与自选的区别

|     | 自选      | 股单               |
|-----|---------|------------------|
| 创建者 | 用户本人    | 平台官方或其他用户        |
| 可编辑 | 可自由增删标的 | 只能关注/取消关注，不可编辑内容 |
| 用途  | 个人持仓跟踪  | 主题投资参考           |

## 股单展示内容

- 股单名称（如「AI 算力核心」「港股机器人概念」）
- 涵盖标的数量
- 整体涨跌幅
- 组合内各标的当日涨跌排列

## 官方股单示例

官方股单由平台运营维护，成分股会随主题变化而调整；用户创建的股单由创建者自行维护。平台定期维护主题股单，覆盖热门投资方向：

- AI 算力核心
- AI 软件应用
- 港股机器人概念
- 港股光伏
- 港股创新药
- 内存与存储
- 韩国市场相关股
- SpaceX 相关 / OpenClaw 相关

## 操作

- **关注**：点击股单详情页的关注按钮，后续在「关注」Tab 可看到该股单动态；关注股单与关注个股不同——关注股单后，该股单整体动态（调仓、成分股异动）出现在「动态 → 关注」Feed，关注个股则在自选列表跟踪单只行情
- **添加至自选**：在股单内找到感兴趣的标的，单独添加至个人自选
- **排序**：可按「添加时间」或「当日涨跌」对股单内标的排序
- **创建自定义股单**：部分账户等级支持，在股单页面点击右上角创建入口操作
