# 邀请关系

邀请关系是长桥平台记录「谁邀请了谁」的核心机制，直接决定邀请活动奖励能否正确归因和发放。

平台维护三种独立的邀请关系：

| 关系类型 | 触发时机 | 说明 |
|---------|---------|------|
| **注册邀请关系** | 被邀请人完成注册时 | 记录谁介绍该用户来到平台 |
| **开户邀请关系** | 被邀请人完成 KYC 开户前 | 记录谁邀请该用户开户，用于开户奖励归因 |
| **入金邀请关系** | 被邀请人首次入金前 | 记录谁邀请该用户入金，用于入金邀请奖励归因 |

三种关系相互独立，可以对应同一位邀请人，也可以不同（例如注册时没有填邀请码，但开户前补绑了邀请人）。

## 如何绑定邀请关系

### 方式一：注册时通过邀请链接或填写邀请码

这是最常见的方式。系统在注册完成时**立即绑定注册邀请关系**，开户邀请关系和入金邀请关系由系统后续自动同步到同一位邀请人。

**前置条件：**
- 被邀请人为平台新用户（未曾注册）
- 邀请人拥有有效的专属邀请码

**操作步骤：**
1. 邀请人在 APP「活动中心」或个人主页获取专属邀请链接 / 邀请码
2. 被邀请人通过邀请链接打开 APP，或在注册页手动填写邀请码
3. 完成注册后，系统绑定注册邀请关系，开户/入金邀请关系随后自动同步

**注意：** 注册完成后，注册邀请关系无法再更改。开户邀请和入金邀请在未完成对应步骤前，仍可通过客服协助补绑或更换邀请人。

### 方式二：注册后补绑邀请人（仅限未开户或未入金前）

适用于注册时未填写邀请码、事后想补绑的用户。

**开户邀请关系补绑条件：**
- 被邀请人尚未完成 KYC 开户（账户状态为未开户）
- 邀请人的邀请码有效

**入金邀请关系补绑条件：**
- 被邀请人尚未完成首次入金
- 邀请人的邀请码有效

一旦完成开户或入金，对应的邀请关系即锁定，**无法再更改或补绑**。

## 奖励归因：哪种邀请关系决定谁拿奖励

当三种邀请关系绑定的是**不同的邀请人**时，各任务奖励按以下优先级归因：

| 任务类型 | 奖励归属优先级 |
|---------|-------------|
| 注册任务 | 注册邀请人 |
| 开户任务（KYC） | 开户邀请人 → 若无则回退至注册邀请人 |
| 入金 / 转仓 / 净资产任务 | 入金邀请人 → 若无则开户邀请人 → 若无则注册邀请人 |

**注意：** 注册邀请关系需在 **2021 年 4 月 28 日之后**建立，才能参与奖励归因。更早的历史注册绑定不触发活动奖励计算。

## 香港账户与新加坡账户的邀请差异

- **不支持跨账户邀请**：长桥香港账户客户不能邀请长桥新加坡账户客户，反之亦然
- **新加坡账户暂无邀请码机制**：新加坡账户目前仅支持通过邀请链接绑定邀请人；若被邀请人已注册成功，则无法再补绑邀请人

## 绑定失败的常见情况

以下情况会导致邀请关系绑定**失败**：

| 情况 | 说明 |
|------|------|
| 邀请自己 | 不能使用自己的邀请码绑定 |
| 互相绑定 | 若 A 已邀请 B，则 B 不能反过来邀请 A |
| 已开户绑开户邀请 | 被邀请人已完成 KYC 开户后，无法绑定或变更开户邀请人 |
| 已入金绑入金邀请 | 被邀请人已完成首次入金后，无法绑定或变更入金邀请人 |
| 邀请码无效 | 邀请码不存在或已失效 |

## 绑定变更与关系纠纷答疑

**Q：注册时没填邀请码，现在还能绑定邀请人吗？**

A：注册邀请关系无法补绑。但如果您尚未完成开户（KYC），开户邀请关系仍可补绑；如果您尚未完成首次入金，入金邀请关系也仍可补绑。请联系客服，提供邀请人的邀请码，客服会协助核查并处理。补绑成功后，对应的邀请活动奖励才能正常发放。

**Q：我注册时填了邀请码，为什么开户奖励没有发给我的邀请人？**

A：请确认以下几点：（1）注册时是否成功绑定了邀请关系——可在 APP「我的 → 邀请码」查看您的邀请人信息；（2）邀请活动是否仍在进行中，活动结束后完成的任务通常不计入奖励；（3）奖励发放需要 1-3 个工作日，请耐心等待。如以上均正常但仍未收到奖励，请联系客服核查。

**Q：我的邀请人信息显示的是 A，但我的邀请码是 B 给的，正确吗？**

A：这种情况可能是注册邀请与开户/入金邀请绑定了不同的人，属于正常情况。三种邀请关系各自独立，奖励会分别按对应关系归因。如您认为有误，请联系客服说明情况。

**Q：我发给朋友的邀请码，朋友说无法使用，是什么原因？**

A：邀请码无法使用的常见原因：（1）朋友已经是平台注册用户，无法作为被邀请人；（2）邀请码输入有误，注意区分大小写及易混淆字符；（3）活动已结束，邀请链接过期。请让朋友检查账户状态，确认是全新注册，并重新通过邀请链接进入。

**Q：被邀请人开户后，邀请关系还能改吗？**

A：开户完成后，开户邀请关系即锁定，**无法更改**。入金邀请关系在首次入金完成后也同样锁定。如确实存在绑定错误（如注册时误填了别人的邀请码），请联系客服说明情况，客服会评估是否可以人工处理，但不保证一定可以更改。

**Q：我邀请了一个朋友，但他后来注销了账户，我的邀请关系还在吗？奖励会受影响吗？**

A：邀请关系记录在系统中，账户注销不会自动删除已绑定的邀请关系。但注销账户通常意味着相关任务进度也会受影响，已发放的奖励根据活动规则可能会被回收或取消。具体情况请联系客服核查。

**Q：同一个用户为什么会有三种不同的邀请关系？**

A：三种邀请关系分别对应用户生命周期的不同阶段——注册、开户（KYC）、首次入金。不同阶段的活动奖励可能归因到不同的邀请人，因此分开记录。对于大多数通过邀请链接注册的用户，三种关系通常绑定的是同一位邀请人。

**Q：邀请人信息（昵称、电话）显示不全，是正常的吗？**

A：是正常的。出于隐私保护，系统展示邀请人信息时会对手机号进行脱敏处理（如显示 138****5678）。如需确认邀请人身份，可联系对方核对。

**Q：活动要求「通过邀请注册」，但系统显示我没有邀请关系，怎么办？**

A：请先确认注册时是否通过邀请链接打开 APP 或填写了邀请码。如确认已操作但系统未记录，请联系客服，提供邀请人的邀请码及您的注册时间，客服会协助核查注册时的绑定情况。

**Q：被邀请人用了我的邀请码，但奖励算到了别人头上，怎么回事？**

A：可能原因：（1）被邀请人此前已通过其他渠道绑定了另一位邀请人的邀请关系；（2）被邀请人在注册时通过其他链接进入，实际绑定的是另一个邀请码；（3）不同任务的奖励归属不同类型的邀请人——例如注册和开户奖励可能分别归属不同邀请人（见「奖励归因」章节）。邀请关系以系统记录为准，如有争议请联系客服核查。

## 绑定规则与违规判定

- 邀请码绑定以系统记录为准，口头约定或截图均不作为依据
- 管理员可人工修改开户或入金邀请关系（限被邀请人尚未完成对应步骤），普通用户如需修改需联系客服由后台处理
- 所有邀请关系均有变更记录，客服可查询历史变更情况
