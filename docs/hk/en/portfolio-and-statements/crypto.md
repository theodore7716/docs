# 加密货币

加密货币页展示主流数字货币的实时行情数据，通过 HashKey Exchange 提供交易服务。

## 页面入口

底部导航「市场」→ 顶部切换「加密货币」标签。

## 焦点卡片

页面顶部以大卡片形式重点展示 BTC（比特币）和 ETH（以太坊）的实时行情：

| 字段        | 说明              |
|-----------|-----------------|
| 币种名称 / 代码 | 加密货币全称和交易代码     |
| 最新价格      | 以 USD 计价的实时价格   |
| 24h 涨跌幅   | 过去 24 小时内的涨跌百分比 |
| 走势图       | 近期价格趋势缩略图       |

点击卡片进入该币种详情页；点击「市场概览」查看整体加密货币市场数据。

## 币种列表

列表展示更多可交易币种，每行包含：

| 字段      | 说明                             |
|---------|--------------------------------|
| 名称 / 代码 | 币种全称和交易对代码（如 APT/USD）          |
| 交易所     | 提供交易服务的交易所（如 HashKey Exchange） |
| 最新价     | 以 USD 计价的实时价格                  |
| 涨跌幅     | 24h 涨跌百分比                      |
| 走势图     | 近期价格趋势缩略图                      |

## 可执行操作

- 点击 BTC / ETH 焦点卡片 → 进入单币种行情详情
- 点击「市场概览」→ 查看整体加密货币市场
- 点击列表中任意币种 → 进入该币种详情页及交易入口

> 加密货币价格波动较大，投资前请充分了解相关风险。
