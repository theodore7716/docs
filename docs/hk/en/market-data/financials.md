# 财务分析

财务分析页面提供上市公司三大财务报表的深度解读，涵盖利润表、资产负债表、现金流量表及股息数据，并支持与同行业公司对比。帮助投资者快速了解公司盈利能力、资产结构和现金状况。

入口：长桥 App - 个股详情页 - 财务

## 财报速览

页面顶部提供最新一期财报的核心数据摘要，包含公布值、同比变化及与市场预测值的对比情况。

| 指标 | 说明 |
|------|------|
| 营业收入 | 当期总收入，显示公布值与预测值对比 |
| 息税前利润 | 扣除利息和税前的利润（EBIT） |
| 每股收益 | EPS，显示实际值与分析师预期值对比 |

- 可通过图表直观对比「公布值」与「预测值」的走势
- 点击「更多」查看完整财报内容

> **一图读懂财报**：页面顶部提供当季财报的图文摘要，快速掌握公司财务状况的整体概况。

## 利润表

展示公司盈利情况，可按单季报或年报查看，支持历史多期数据横向对比。

| 指标 | 说明 |
|------|------|
| 每股收益（EPS） | 每股对应的净利润 |
| ROE | 净资产收益率，衡量股东权益回报 |
| 营业收入 | 当期实现的总营收 |
| 净利润 | 扣除所有费用后的最终利润 |
| 营业利润 | 扣除营业成本和费用后的利润 |
| 毛利率 | 毛利润占营业收入的比例 |
| 净利率 | 净利润占营业收入的比例 |
| 利润含金量 | 经营现金流与净利润的匹配程度 |

- 列表展示近 5 个季度的历史数据，包含环比变动和股价走势
- 点击「更多」可查看更多历史期数据

## 资产负债表

展示公司在特定时点的资产、负债和股东权益状况，帮助判断公司财务健康程度。

| 指标 | 说明 |
|------|------|
| 资产与负债 | 总资产与总负债的规模及变化趋势 |
| 权益乘数 | 总资产与股东权益之比，反映杠杆程度 |
| 每股净资产 | 每股对应的账面净资产价值 |
| 资产周转率 | 资产创造收入的效率 |
| 现金及短投 | 现金及短期投资金额 |
| 存货与应收 | 存货和应收账款情况 |
| 长期投资 | 长期股权及其他投资规模 |
| 净债务 | 有息负债减去现金及现金等价物 |

- 支持查看当前公司在同行业中的资产负债排名（如：43/99）
- 图表展示总资产与总负债的历史走势，以及与营收、净利润的关联分析

## 现金流量表

展示公司实际现金的流入与流出情况，是判断公司造血能力的重要依据。

| 指标 | 说明 |
|------|------|
| 经营现金流 | 主营业务带来的现金净额 |
| 投资现金流 | 投资活动产生的现金净额 |
| 融资现金流 | 融资活动（借款、发债、股权融资等）产生的现金净额 |
| 自由现金流 | 经营现金流减去资本支出后的可自由支配现金 |
| 现金流充裕率 | 经营现金流覆盖负债的能力 |
| 举债与偿债 | 新增借款与还款情况 |
| 资本支出 | 购置固定资产、设备等长期资产的支出 |

- 列表展示近 5 个季度历史数据，含环比变动
- 点击「了解更多经营现金流相关知识」可查看指标说明

## 股息相关

展示公司历史派息记录及股息相关指标，适用于关注现金回报的投资者。

| 指标 | 说明 |
|------|------|
| 股息及股息率 | 每股股息金额及其相对股价的收益率 |
| 股息/净利润 | 派息比例，衡量公司分红意愿 |
| 股息/现金流 | 股息与经营现金流的比值 |
| 股息增长率 | 历史股息的增长趋势 |

> 部分公司暂不派息，此模块将显示「暂无数据」。

## 同行业公司对比

各财务模块均支持与同行业公司横向对比：

- 查看当前公司在行业中的指标排名位置（如：排名 43/99）
- 通过「相关」入口跳转至同类公司的财务数据对比页
- 图表同步展示股价走势与财务指标的相关性

## 数据筛选与切换

- **报告期切换**：支持「单季报」和「年报」两种视图，方便对比季度与全年表现
- **币种说明**：数据默认以公司报告货币展示（如 USD），页面顶部标注当前币种
- **历史数据**：点击各模块的「更多」按钮，可查看更多历史期数据

> 以上行情、资讯及其他数据来自第三方数据源，仅供参考，不构成投资建议。
