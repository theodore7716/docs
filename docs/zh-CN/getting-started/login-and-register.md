# 登录注册

长桥 App 采用手机号一体化设计：**首次登录即自动完成注册**，无需单独注册流程。

## 登录方式总览

| 登录方式 | 入口 | 说明 |
|---------|------|------|
| 手机号 + OTP 验证码 | 默认页面 | 输入手机号发送验证码，首次使用自动注册 |
| 手机号 + 密码 | Select another method | 需提前设置密码 |
| Login ID | Select another method | 使用平台分配的登录 ID + 密码 |
| 邮箱 | Select another method | 使用绑定邮箱 + 密码登录 |
| 支付宝 | 页面底部图标 | 一键授权登录（适用于已绑定的账户） |
| Google | 底部 `···` → 更多 | 使用 Google 账号授权登录 |
| Facebook | 底部 `···` → 更多 | 使用 Facebook 账号授权登录 |

---

## 手机号 + OTP 验证码（默认方式）

1. 打开长桥 App，进入登录页面
2. 点击左侧区号（默认 `+86`）选择手机号归属地区
3. 输入手机号码
4. 勾选下方用户协议复选框
5. 点击「Send OTP」，系统向手机发送一次性验证码
6. 在验证码输入页输入收到的验证码，完成登录。验证码自发送后 5 分钟内有效，过期需重新发送

> **首次登录即自动注册**：若该手机号尚未注册账户，验证码验证通过后系统自动创建账户，无需填写额外信息。

---

## 其他登录方式

点击「Select another method」弹出选择菜单：

- **Phone Number and Password**：使用手机号 + 账户密码登录，需提前在 App 内设置密码
- **Login ID**：使用平台分配的专属登录 ID + 密码。登录后可在「我的 → 账户安全」查看当前账户的 Login ID
- **Email**：使用绑定邮箱 + 密码登录

---

## 第三方快捷登录

登录页底部有两个快捷入口：

- **支付宝图标**：直接点击，通过支付宝授权一键登录
- **`···` 图标**：点击展开更多第三方登录方式，目前支持 **Google** 和 **Facebook**

第三方登录时，系统以授权账号关联的手机号或邮箱匹配已有账户。若有匹配账户则直接登录；若无匹配账户则创建新账号。建议首次使用前确认关联信息，避免产生重复账号。

---

## 选择手机号归属地区

点击输入框左侧区号（如 `+86`）进入国家/地区选择页：

- **搜索**：在顶部搜索框输入国家或地区名称快速定位
- **常用地区**（快速选择）：
  - 中国大陆 +86
  - 澳门 +853
  - 香港 +852
  - 台湾 +886
  - 新加坡 +65
- 完整列表按字母顺序排列，右侧有字母索引可快速跳转

---

## 邀请码

登录页右上角「Invitation Code」为可展开输入框，展开后在手机号输入框下方出现邀请码输入栏。

- 若通过好友邀请注册，请在注册前填入邀请码，确保邀请关系正确绑定
- 邀请码须在**首次注册完成前**填写，注册后无法补填

---

## 用户协议

点击「Send OTP」前需勾选协议，包含：

- **Service Agreement**（服务协议）
- **Privacy Policy**（隐私政策）
- **Third-Party Personal Information Sharing Inventory**（第三方个人信息共享清单）

点击对应文字链接可查看完整协议内容。未勾选协议时，「Send OTP」按钮为置灰不可点击状态。

---


## 账号唯一性、安全规则与手机号变更

- 同一手机号在同一持牌实体下只能对应一个账户；账号以手机号为主要标识，遗失密码可直接用手机号 + OTP 重新登录
- 更换手机号时，在旧号码仍可接收短信期间，进入「我的 → 账户安全 → 手机号管理」操作；若旧号码已无法使用，需联系客服提供身份证明进行人工处理
- 账户处于冻结状态时，登录后会提示账户异常，需联系客服处理
- 长桥不会主动要求您将验证码提供给他人，请勿泄露验证码
- 收不到验证码时，请参阅 [收不到短信验证码](/troubleshooting/sms-code) 排查
- 登录遇到问题，可点击页面右上角「Get Help」直接联系客服
