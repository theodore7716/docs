# 保证金

保证金是持有融资仓位所需的最低资产要求，本页说明三层保证金结构、货币保证金规则及追缴保证金（Margin Call）机制。

## 保证金三层结构

账户设有三层保证金机制：

| 层次 | 缩写 | 含义 | 触发后果 |
| --- | --- | --- | --- |
| 初始保证金 | IM | 开仓时所需的最低保证金，决定购买力上限 | 股权资产 < IM → 购买力用尽，不可新开仓 |
| 维持保证金 | MM | 持仓期间需维持的最低保证金 | 股权资产 < MM → 发出追缴保证金通知 |
| 平仓保证金 | FM | 强制平仓的触发线 | 股权资产 < FM → 可能即时强制平仓 |

## 保证金要求计算

各层保证金要求按持仓加总计算：

保证金要求 = 加总（各持仓市值 × 对应保证金系数）

不同证券的 IM / MM / FM 系数各不相同，具体系数以平台公示为准。各证券保证金系数请以平台实时公示为准，不同市场、不同标的系数存在差异。

## 关键指标定义

股权资产 = 现金余额 + 投资组合总市值，不包括期权市值、债券市值

可用资金（含融资）= 股权资产 - 初始保证金要求 - 冻结资金

最大购买力 = 可用资金 ÷ 初始保证金系数（IM Ratio）

## 购买力计算示例

存入现金 $10,000，账户融资额度 $20,000，目标标的 IM 系数为 30%：

可用资金 = $10,000 - $0（当前无持仓，IM 要求为 0）- $0 = $10,000

理论最大购买力 = $10,000 ÷ 30% ≈ $33,333

但实际受融资额度限制，最多融资 $20,000，因此实际最大购买力 = $10,000 + $20,000 = $30,000

购买后融资结欠 = $20,000，未超过融资额度上限，符合要求。

## 货币保证金

当账户持有的货币与投资标的计价货币不同时（如用港元账户购买美元计价的美股），存在汇率波动风险。平台会对此类跨币种持仓额外收取货币保证金。

### 货币保证金的触发条件

当某种货币的资产净值为负（即该货币资不抵债）时，触发货币保证金计算。

货币保证金 = 该货币负债金额 × 货币保证金比率

### 货币保证金比率

以下比率以平台实时公示为准：

美元（USD）：4%

港元（HKD）：4%

### 货币保证金示例

账户存入 20,000 HKD，未持有美元。

购入价值 2,000 USD 的美股（此时美元持仓为 -2,000 USD，即借入美元）。

之后美股市值下跌至 1,900 USD。

此时美元资产净值 = 1,900 USD（持仓市值）- 2,000 USD（借入成本）= -100 USD，即美元资不抵债 100 USD。

货币保证金 = 100 USD × 4% = 4 USD

### 对账户指标的影响

货币保证金会计入以下计算：

**购买力**：货币保证金作为额外占用，降低可用购买力。

**出金可用金额**：出金时需预留货币保证金，超出部分方可出金。

**追缴保证金**：触发追缴时，货币保证金纳入差额计算。

## 追缴保证金（Margin Call）

当账户股权资产低于维持保证金要求时，平台将发出追缴保证金通知，要求在规定时间内补足差额，否则可能触发强制平仓。

### 追缴触发条件

股权资产 < 维持保证金要求（MM）

### 追缴处理期限

收到追缴通知后，须在 3 个交易日内（含当日）补足差额。

追缴金额 = 维持保证金要求 - 股权资产（高于两者差额）

应对步骤（存入现金或卖出持仓的操作路径）见如何应对追加保证金通知。

### 强制平仓触发条件

股权资产 < 强平保证金要求（FM）时，平台可能即时强制平仓，无需等待 3 个交易日。

以下高风险情况可能不等待追缴期，直接触发强平：

- 末日期权（到期日当天的期权）持仓

- 市场出现极端波动

- 持仓流动性不足，无法正常变现

### 追缴保证金完整示例

账户存入现金 $10,000，同时持有 A 股票市值 $5,000（IM 30% / MM 25%）。

使用融资购入 B 股票，买入价 $25,000（IM 50% / MM 45%）。

之后 B 股票市值下跌至 $19,500。

此时各项指标：

股权资产 = $10,000（现金）+ $5,000（A）+ $19,500（B）- $25,000（融资结欠）= $9,500

初始保证金要求 = $5,000 × 30% + $19,500 × 50% = $1,500 + $9,750 = $11,250

维持保证金要求 = $5,000 × 25% + $19,500 × 45% = $1,250 + $8,775 = $10,025

股权资产 $9,500 < 维持保证金要求 $10,025，触发追缴。

追缴金额需补至初始保证金水平：$11,250 - $9,500 = $1,750

如客户选择卖出证券，所需卖出的市值不多于初始保证金要求与股权资产的差额除以初始保证金系数（IM Ratio）：

所需卖出市值 ≤（初始保证金要求 - 股权资产）÷ IM Ratio

实际追缴金额及截止时间请以账户内通知为准。

注：实际追缴金额可能较高，视乎股票性质和公司政策而定。
