# 跨账号划转

同一长桥 ID 跨券商和不同 ID 跨券商资金划转的授权、操作、到账时间及限额。

## 同一账号跨券商划转

为加速实现“AI网券商”的战略愿景，Longbridge 正对账户系统进行全面升级。 此次升级不仅提升开户审核效率，更将为未来一系列智能化功能与账号关联。给用户带来更优的体验。 为确保系统平稳切换，优化期间资金划转功能暂时下架，感谢您的使用。

**资金划转黑名单限制：** 同一账号及非同一账号跨券商资金划转均需满足：客户不在长桥资金划转黑名单内。如划转申请被拒，请联系客服确认账户状态。

### 授权流程

首次使用需先完成授权：

1. 长桥 App - 资产 - 全部功能 - 资金划转 - 跨券商账户资金划转

2. 发起授权申请，阅读授权协议（需滑到最底部），确认个人信息后点击「立即授权」

3. 授权成功后即可进行资金划转

![](./images/OHa2bbirFoMaB5xD0mAj3SaKpdh.png)

跨券商资金划转授权流程

如系统检测到香港和新加坡账户信息不一致，需人工审核，等待授权结果即可。如两边证件不一致，需在对应柜台提交更新证件。

### 划转操作

授权成功后，选择转入账户和转出账户，填写划转金额，确认信息后提交。

![](./images/MFn1bpj7CoKrOux4UJTjEzcgpxg.png)
![](./images/NJ9NbBQHhoxRr4xuGCfjxga5pnd.png)

跨券商资金划转表单

- H 开头的证券账号为长桥香港账户

- SG 开头的证券账号为长桥新加坡账户

### 到账时间

16:00 之前申请的，预计 1-2 个工作日到账；之后申请的，预计 1-3 个工作日到账。由于香港和新加坡当地节假日不同，具体工作日以转入券商所在地为准。

### 划转限额

单笔转账融资划转金额不超过 10 万 HKD。

### 撤销划转

资金划转处于「已提交」状态时，可在划转记录详情页点击撤销。

### 进度查询与余额通限制

- 划转成功后会收到推送和邮件通知，也可在划转记录详情页查看进度

![](./images/RS2kbot2dooJu7xvpAnjgsgGpqf.png)
 - 如账户资金在余额通中，资金划转额度会显示为 0，需先赎回余额通或等待资金交收完毕

## 非同一账号跨券商划转

适用于在两个不同长桥 ID 下分别拥有长桥香港账户和长桥新加坡账户的用户。

例如：长桥香港账户的长桥 ID 为 3456789，长桥新加坡账户的长桥 ID 为 3987654。两个账户需分别在不同设备/APP 下登录，不能在 APP 资产页面直接切换。

### 操作流程

1. 联系客服申请人工预关联，提供两个账户的证券 ID 信息（无需提供证件信息）

2. 预关联完成后，在 APP 内提交授权申请

3. 授权申请提交后需人工审核，审核通过后即可进行资金划转

4. 在长桥香港发起的划转需用香港账号确认，在长桥新加坡发起的需用新加坡账号确认
