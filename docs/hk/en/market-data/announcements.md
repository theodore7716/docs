# 公告与讨论

个股详情页提供「资讯」「讨论」「公告」三个 Tab，聚合了与该股票相关的公司公告、社区讨论和市场资讯，帮助投资者快速掌握最新动态。

入口：长桥 App - 个股详情页 - 顶部 Tab 切换

## 公告 Tab

展示上市公司发布的官方公告，按时间倒序排列。

### 公司日程

独立模块，汇总近期重要事项提醒，包括：

- **财报发布**：预计财报披露日期及对应财季（如「2026 财年第 1 季度业绩披露」）
- **分配方案**：派息安排，显示每股派息金额及币种
- **登记日 / 除权日**：分红相关关键日期

### 公告分类筛选

公告列表支持按类型筛选，分类包括：

| 分类 | 说明 |
|------|------|
| 全部 | 显示所有类型公告 |
| 重大事项 | 重大合同、人事变动、重组并购等 |
| 财务报告 | 年报、中报、季报等财务披露 |
| 交易相关 | 股票回购、增减持、配股等 |
| 上市文件 | 招股书、上市资料等 |

每条公告显示发布日期和公告标题，点击可查看原文。

## 讨论 Tab

聚合长桥社区中关于该股票的讨论内容。

### 涨跌预测

页面顶部展示社区涨跌投票情况：

- 显示「猜涨」和「猜跌」的当前比例（如猜跌 88%、猜涨 12%）
- 可参与投票，表达对该股短期走势的判断
- 显示奖池金额及开奖倒计时

### 关注与表态

显示该股票的关注人数和社区表态总数（如「56.1k 人关注 · 2.4k 条表态」），反映该股的社区热度。

### 帖子列表

支持按「最新」和「热门」切换排序。每条帖子显示：

- 发帖人昵称及身份标签（如「评论员」「收益排行」等特殊标识）
- 发帖时间
- 帖子正文（超长内容折叠显示，点击「展开」查看全文）
- 帖子中可包含@提及的股票代码，点击可跳转至对应个股页面

### 达人视图

切换至「达人视图」可单独查看认证达人、评论员发布的内容，过滤普通用户帖子，聚焦高质量分析。

## 资讯 Tab

聚合来自各大财经媒体和资讯平台的新闻报道。

每条资讯显示：

- 新闻标题（含中英文来源）
- 来源媒体名称（如虎嗅、新浪财经、金融界、Tip Ranks 等）
- 发布时间（如「1 小时前」「37 分钟前」）

资讯内容涵盖：公司最新动态、行业分析报告、机构评级调整、市场异动速递等与该股票相关的信息。
