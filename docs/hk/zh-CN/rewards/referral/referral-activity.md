# 邀请活动

邀请活动（Invite Activity）是 LongBridge
平台推出的老带新推广活动。现有用户通过分享专属邀请码或邀请链接，邀请新用户注册、完成开户（KYC）并满足指定任务条件后，邀请人（老用户）与被邀请人（新用户）均可获得平台奖励。邀请活动是平台最核心的用户增长活动之一，奖励丰厚，支持阶梯式累计奖励。

## 参与条件

- 邀请人需为已注册并完成实名认证（KYC）的 LongBridge 账户
- 被邀请人需为平台新用户（未注册或未绑定邀请关系）
- 参与资格以账户所属地区及持牌人信息为准，部分地区可能不支持参与
- 活动期间有效，活动结束后邀请关系仍存在但可能不计入奖励

## 任务类型

- **注册任务**：被邀请人通过邀请码/链接完成新账户注册
- **开户任务（KYC）**：被邀请人完成实名认证，即身份证/护照验证及账户审核
- **入金任务**：被邀请人首次向账户存入指定金额
- **交易任务**：被邀请人在活动期间完成指定笔数或金额的证券交易
- **综合任务**：以上多项任务的组合，需全部完成才能解锁最高奖励

## 参与步骤

1. 邀请人在 APP「活动中心」或个人主页找到专属邀请码/邀请链接
2. 将邀请码或链接分享给好友（微信、短信、社交媒体等渠道均可）
3. 好友通过链接下载 LongBridge APP，或在注册页填写邀请码
4. 好友完成注册，系统自动绑定邀请关系
5. 好友按活动要求完成后续任务（开户/deposit/交易）
6. 任务完成后系统触发奖励发放流程
7. 邀请人和被邀请人均可在「活动中心 → 我的奖励」查看奖励状态

## 任务完成判定

- **注册绑定**：被邀请人使用邀请链接或在注册流程中填写邀请码，系统在注册完成时自动记录邀请关系
- **KYC 完成**：审核通过后系统自动标记
- **入金到账**：资金实际到达交易账户，处理中状态不计入
- **交易完成**：以「最终有效成交」为准，撤单/废单不计入交易任务进度

## 奖励类型

- **现金奖励**：直接发放至 LongBridge 账户可用余额
- **股票奖励**：指定股票若干股，发放至证券账户
- **交易券**：用于抵扣交易手续费
- **体验金**：有使用期限和条件限制的模拟资金
- **现金券**：满足一定条件可提现或使用的优惠券

## 发放时机

被邀请人完成指定任务后，系统通常在 1-3 个工作日内触发奖励发放。阶梯奖励按每完成一位好友任务后即时结算，无需等活动结束。部分活动规定任务完成后
T+1 工作日发放。具体发放时机以活动详情页说明为准。

## 奖励状态说明

| 状态    | 含义                | 用户操作             |
|-------|-------------------|------------------|
| 待发放   | 任务已被系统确认，奖励正在排队处理 | 无需操作，等待系统处理      |
| 发放中   | 系统正在将奖励发送至账户      | 等待，通常几分钟至数小时完成   |
| 已发放   | 奖励已成功到账           | 可在账户余额或持仓中查看     |
| 锁定中   | 奖励已记录但处于锁定期，暂不可用  | 等待锁定期结束自动解锁      |
| 已解锁可用 | 锁定期结束，奖励已可正常使用    | 正常使用奖励           |
| 发放失败  | 奖励发放未成功           | 联系客服处理           |
| 已取消   | 奖励因规则不符或活动调整被取消   | 通常不可恢复，可联系客服确认原因 |

## 阶梯奖励

邀请活动常设有阶梯奖励机制，邀请人奖励随成功邀请人数递增。阶梯计算以活动期间内成功完成任务的被邀请人总数为准，具体阶梯金额和档位以活动详情页为准。

单个用户在同一邀请活动中可获得的奖励总数通常有上限，超出上限部分不予发放，具体上限以活动详情页说明为准。

## 锁定期（Lockup）

部分奖励（尤其是股票或较大金额现金）发放后进入锁定期，状态显示为「锁定中（Lockup）」，到期后自动变为「已解锁可用（Used）」。锁定期通常为
30-180 天，锁定期内不可提现、不可交易，但计入账户总资产展示。

>
锁仓期的完整机制说明（计算规则、违约后果等）请参见[奖励系统 · 锁仓期机制](/rewards/rewards-overview#锁仓期机制)。

## 留存任务

部分邀请活动要求被邀请人完成**留存任务**才能触发奖励。活动页面会展示以下留存相关节点：

- **留存进行中**：被邀请人已开始留存，但尚未满足留存天数/资产要求
- **留存完成**：被邀请人成功完成留存条件，邀请奖励可触发
- **留存失败**：被邀请人留存中断（如资产跌破门槛、提前出金等），该阶段奖励不发放

若活动设有留存任务，邀请人须等被邀请人完成留存后才能收到对应奖励。

## 邀请排名奖励

部分邀请活动设有邀请排行榜，统计活动期间内各邀请人成功邀请好友的数量。排名靠前的邀请人在活动结束结算后可额外获得排名奖励（在普通邀请奖励之上叠加）。排名奖励的档位和金额以活动详情页为准，通常在活动结束后
1-3 个工作日内发放。

## 邀请绑定与奖励发放答疑

**Q：我邀请了好友，好友也完成了开户，为什么我还没收到奖励？**
A：奖励发放通常需要 1-3 个工作日，请耐心等待。您可以在「活动中心 → 我的奖励」查看奖励的当前状态（待发放/发放中/已发放）。如超过
3 个工作日仍未到账，请联系客服并提供好友的注册信息以便核查。

**Q：我发给朋友的邀请链接，朋友说无效/邀请码不对，是怎么回事？**
A：邀请码无效或无法绑定通常有以下原因：（1）好友已经注册过 LongBridge
账户，不属于新用户；（2）好友注册时没有通过您的邀请链接，也没有在注册页输入邀请码；（3）活动已结束，邀请链接过期。请确认好友是全新注册，并在注册第一步就通过您的链接或填写邀请码。注册邀请关系一旦注册完成即绑定，事后无法更改；但开户/入金邀请关系在好友尚未完成开户或入金前，仍可联系客服补绑。

**Q：好友还没完成开户（KYC），我的邀请还有效吗？有时间限制吗？**
A：邀请关系在好友注册时即完成绑定，但奖励发放取决于好友完成所有指定任务。若好友注册后迟迟未完成 KYC
或其他任务，通常需要在活动有效期内完成方可触发奖励。活动结束后完成的任务一般不再计入奖励资格。建议提醒好友尽快完成开户流程，具体截止时间以活动详情页为准。

**Q：阶梯奖励是怎么算的？我邀请了 5 个人，总共能拿多少？**
A：阶梯奖励按成功邀请的好友人数累计计算。例如活动设定「邀请第 1 人得 50 港币，第 2-3 人每人 100 港币，第 4 人起每人 150
港币」，则邀请 5 位好友的总奖励为：50 + 100 + 100 + 150 + 150 = 550 港币。每位好友完成任务后系统独立结算，无需等所有好友都完成。具体阶梯金额以活动详情页为准。

**Q：我的奖励状态显示「发放失败（Failed）」，怎么办？**
A：奖励发放失败最常见的原因包括：（1）账户未完成实名认证（KYC）；（2）账户存在异常或被冻结；（3）奖励类型与账户类型不匹配；（4）系统技术问题。请先检查您的账户状态是否正常、KYC
是否完成，然后联系客服提供奖励 ID 进行核查，客服会协助重新触发发放或给出解决方案。

**Q：奖励显示「锁定中（Lockup）」，什么时候能用？**
A：锁定中状态表示奖励已成功记录到您的账户，但处于锁定期内暂时无法使用。锁定期结束后系统会自动将状态变更为「已解锁可用（Used）」，届时您可以正常使用该奖励。锁定期长度以活动规则为准，您可以在「活动中心 →
我的奖励」查看预计解锁日期。

**Q：被邀请人（我的好友）能获得什么奖励？**
A：通过邀请码注册的新用户通常也能获得欢迎礼包，包括现金券、交易券或体验金等。具体奖励内容以活动详情页中「新用户奖励」部分的说明为准。被邀请人奖励与邀请人奖励独立发放，互不影响。欢迎礼包中的体验金、交易券等有使用期限，过期未使用将自动取消，请在有效期内使用。

**Q：我朋友被我邀请，完成了所有任务，但奖励显示已取消（Cancel），是什么原因？**
A：奖励被取消通常有以下原因：（1）被邀请人账户被判定为异常，如与已有账户存在关联（同一设备、同一持牌人名下）；（2）活动期间规则调整导致部分奖励不符合新规则；（3）自我邀请或违规邀请行为被系统识别；（4）被邀请人账户后续被注销或永久限制。请联系客服，提供您好友的账户信息和完成任务的截图，客服会协助核查具体取消原因。

**Q：我的活动奖励一直是「待发放」状态，是被风控了吗？**
A：账户处于风控审核期间，活动进度可能暂停计算，奖励保持待发放状态，风控解除后会恢复正常处理。除风控外，奖励排队处理也可能造成短期停留在待发放状态（通常
1-3 个工作日内完成）。如超过 3 个工作日仍无变化，请联系客服并配合账户验证流程。

**Q：活动进行中规则调整了，已触发的奖励会受影响吗？**
A：已触发发放的奖励通常不受规则调整影响。尚未触发的奖励以调整后的最新规则为准。规则变更会在活动页面公告，建议定期查看活动详情页。

**Q：系统故障期间完成了任务，但活动进度没有记录，怎么办？**
A：如遇平台系统故障导致活动进度未正常记录，请保留故障期间的交易/操作截图，提交客服工单说明情况。技术团队会核查日志并进行数据补录（通常
3-5 个工作日）。补录需要用户提供证明材料，平台不保证所有情况均可补录。

## 邀请合规要求与违规处理

- 已有 LongBridge 账户的用户不能作为被邀请人参与
- 同一设备、同一 IP 地址注册的多个账户可能被判定为异常，导致奖励不发放
- 自我邀请（用自己的邀请码注册新账户）属于违规行为，奖励将被取消
- 同一持牌人（真实个人）名下无论有几个账户，被邀请次数只计 1 次，利用子账户刷取邀请奖励属于违规行为
- 注册邀请关系在注册完成时绑定，之后无法更改；开户邀请关系和入金邀请关系在用户尚未完成对应步骤（开户/首次入金）前仍可补绑，完成后锁定不可更改
- 同一被邀请人在同一阶段（开户/入金）只能对应一位邀请人；在完成该阶段前可以更换邀请人，完成后锁定不可更改
- 请务必让好友通过您的专属链接注册，或在注册第一步填写邀请码，中途返回重填可能导致绑定失败
- 所有活动时间以香港时间（HKT，UTC+8）为准
- 奖励查看路径：APP「活动中心 → 我的奖励」
