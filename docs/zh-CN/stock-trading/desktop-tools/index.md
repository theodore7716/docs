# 桌面端工具

长桥桌面端提供专业级的交易工具，包括高级图表、市场深度数据和自定义布局。这些工具帮助你更高效地分析行情、做出交易决策。

## 核心工具介绍

| 工具 | 用途 | 何时使用 |
|------|------|---------|
| **图表交易** | 通过 K 线图进行技术分析和快速下单 | 需要详细技术分析、多时间周期对比 |
| **市场深度** | 实时查看买卖盘口、委单深度 | 需要了解市场流动性、下单前观察盘口 |
| **组件布局** | 自定义桌面端界面排布 | 根据个人交易习惯优化工作区 |
| **快捷键** | 键盘快速操作 | 需要提高操作效率、快速应对行情 |
